## 一、卷一总定位

### 卷一核心主题

**一个被公司榨干的程序员，在被迫失去旧人生后，借一条鱼重新夺回命运。**

### 卷一核心任务

1. 完成陆沉从"打工人"到"正式赛手"的第一次身份转变
2. 完成陆沉与红焰从"偶然相遇"到"初步共斗"的绑定
3. 完成公司线第一轮清算：张海明 / 赵鹏阶段性败北
4. 完成蓝鲸杯从"民间幻想"到"真实战场"的落地
5. 打开更大世界观：自然会、斗王计划、父亲旧线、母亲病线开始升温

### 卷一最终终点

> **陆沉以黑马身份拿下蓝鲸杯冠军，正式脱离蓝星旧人生，搬入基地，获得龙门杯市级赛资格；红焰完成卷一第一次关键成长；张海明 / 赵洋阶段性败退，但更大的赛场和更深的真相开始浮出水面。**

## 三、阶段详纲

---

### 阶段1：报名截止

**时间**：2026年3月2日（周一）— 2026年3月6日（周五）

**已有基础**：

崩溃（周一凌晨）→ 犹豫（周二）→ 爆发（周三）→ 羁绊（周三晚）→ 被利用（周四）→ 觉醒（周五）→ 报名（截止前）

**主冲突**：

- **现实线**：陆沉与张海明彻底撕裂，旧人生崩盘
- **内心线**：陆沉还在"认命"与"换条路试试"之间摇摆
- **新主线引爆**：红焰出现，蓝鲸杯不再只是陈海嘴里的幻想

**情绪目标**：

- 让读者对陆沉产生强烈代入：压抑、憋屈、爽快、发冷
- 让读者完成对红焰的第一次"上头"
- 让公司线的恶心值拉满，逼着读者想看陆沉反杀

**赛事目标**：

- 只做"赛事入口投放"，不正式开赛
- 建立：蓝鲸杯存在感 / 红焰价值感 / 周叔/渔具店/基地雏形 / "报名"成为当前第一目标

**伏笔回收目标**：

- 小回收：陆沉第一次反抗张海明 / 红焰从"异常鱼影"变成正式命名个体 / 周叔出场
- 强化埋线：红焰真实身份 / 周叔看破不说破 / 陈海和父亲旧线 / 张海明做局 / 赵鹏背刺 / 自然会远线

**阶段高潮**：第7章《英雄》—— 全体会议上：赵鹏抢功、张海明反咬、剪辑录音公开处刑

**阶段断章点**：

> 会议结束后，陆沉彻底明白：昨天回去，不是为了翻盘，而是为了让别人把局做完整。下一步必须是：**报名 / 离职 / 转向比赛线。**

---

### 阶段2：备赛立身

**时间**：2026年3月7日 — 2026年3月15日

**主冲突**：

- **现实线**：陆沉必须真正完成"离开公司"这件事，而不是只停留在心理决裂
- **比赛线**：他和红焰都还不够格，想参赛就得先补完"选手资格、鱼认证、训练基础、队伍结构"
- **资源线**：钱、住处、基地、鱼缸、训练节奏都要落地

**情绪目标**：

- 从"被动崩溃"转成"主动站起来"
- 从"爽完之后的空"转成"开始做事的硬"
- 让读者感到：陆沉不是逃进钓鱼，而是在认真搭建新人生

**赛事目标**：

- 完成蓝鲸杯报名
- 完成红焰基础认证 / 评级 / 检录准备
- 完成基地正式启用
- 建立初步训练体系
- 明确队伍形式：个人 / 双人 / "沉海"关系重新定义
- 让陆沉真正学会"赛手怎么训练鱼、怎么用声波指令、怎么理解场地"

**伏笔回收目标**：

- 回收/强化：蓝鲸杯报名 / 离职状态推进 / 基地落地并投入使用
- 强化：陈海与父亲关系再强化一点，但不说透 / 周叔知道很多但只给半步信息

**阶段高潮（方案B优先）**：

> 陆沉正式按下蓝鲸杯报名键，同时完成公司手续断裂，两个世界正式切开。

**阶段断章点**：

> 蓝鲸杯浅圳赛区抽签结果出来。真正的比赛，不再是想象，而是明天就要下场。

---

### 阶段3：入围杀场

**时间**：2026年3月16日起（浅圳赛区开打）

**主冲突**：

- **比赛线**：陆沉第一次以真正赛手身份下场，面对规则、场地、陌生对手、临场指挥，而不是河边野钓
- **能力线**：第一阶段鱼色的信息差，必须真正转化成实战收益
- **心理线**：他不能再用"我是新手"给自己找退路

**情绪目标**：

- 从"训练期不稳"转成"第一次上场的紧张 + 爽感"
- 让读者看到：陆沉不是直接无敌，而是边学边打、边打边成
- 让红焰第一次真正打出"鲤鱼王"的观感

**赛事目标**：

- 通过浅圳赛区入围赛
- 明确蓝鲸杯真实战斗观感：1团队vs1团队vs1团队vs1团队 / 每团队最多4条鱼 / 超级计算机保护 / 场地主题/混战/团队调度

**伏笔回收目标**：

- 回收：鱼色能力第一次在正式比赛里兑现价值 / 报名不再是悬念
- 强化：赵鹏背刺后陆沉更珍惜"靠自己赢" / 自然会第一次注意到陆沉/红焰

**阶段高潮**：

> 蓝鲸杯浅圳赛区决定晋级资格的一场混战——陆沉靠"看穿鱼色 + 初步指挥 + 红焰爆发"，完成第一次正式逆转，拿到社会晋级资格。

**阶段断章点**：

> 陆沉以为自己只是赢了一场入围赛，实际上他和红焰已经进入更大的视线。

---

### 阶段4：正赛成名

**时间**：4月11日开始的蓝鲸杯正赛阶段

**主冲突**：

- **比赛线**：陆沉从"赛区黑马"进入真正高手区
- **对手线**：赵洋、张海明这条线必须正式碰撞
- **资源线**：红焰一个核心不够，团队体系、指挥、场地理解都要升级
- **世界观线**：自然会/赛事方/父亲旧线开始向主线靠拢

**情绪目标**：

- 从"小人物侥幸突围"升级成"黑马真的开始成名"
- 让读者体验：越往上打，越不是拼鱼狠，而是拼理解、拼调度、拼胆子
- 让红焰从"怪鱼"升级成"真正开始让圈内人记住的鱼"

**赛事目标**：

- 打穿蓝鲸杯正赛前几轮
- 正面对上赵洋体系
- 让陆沉第一次真正意识到"赛手之间的差距，不是有鱼和没鱼，而是会不会用鱼"

**伏笔回收目标**：

- 回收：周涛/刘健在现实线里形成真正支援 / 赵鹏背刺线完成阶段性清算
- 强化：张海明现金流/贿赂领导真实动机开始显影 / 陈海与父亲旧关系揭开第一层 / 周叔不说破的原因开始给一点答案 / 红焰身份和旧伤来历继续抬升

**阶段高潮**：

> 陆沉 vs 赵洋（或张海明体系）的正面碰撞——这不仅是比赛，还是"旧人生的脸"。

**阶段断章点**：

> 赢下对手之后，陆沉发现自己赢的不是全部——真正想看他的人，终于出现了。

---

### 阶段5：冠军与代价

**时间**：蓝鲸杯决赛阶段 + 卷尾收束

**主冲突**：

- **终极比赛线**：陆沉必须证明他不是一时运气，而是真能站到冠军位置
- **内在线**：他要回答卷一真正的问题——**我是逃离旧人生，还是第一次真正握住自己的命运？**
- **世界观线**：赢下冠军后，他不可能继续只是个民间选手

**情绪目标**：

- 卷尾要做到：热血 / 爽 / 感动 / 但不能只爽，要有"赢了以后代价也更大"的抬升感

**赛事目标**：

- 拿下蓝鲸杯冠军
- 获得龙门杯市级赛资格
- 正式建立"陆沉 + 红焰"的黑马名声

**伏笔回收目标**：

- 阶段性回收：比赛主线正式跑顺 / 正式离职流程完成 / 基地从临时落脚点变成真正的训练据点
- 半回收/抬升：红焰真实身份给出更强指向但不彻底讲透 / 母亲病线要在卷二情感炸点前置 / 自然会第一次正式接触或留下名片 / 斗王计划从"影子"升级成"有人明确提起，但主角还不懂"

**阶段高潮**：

> 蓝鲸杯决赛——陆沉和红焰完成卷一最强共斗，真正把"鲤鱼王"这个名号狠狠干出来。

**阶段断章点（推荐优先选1）**：

1. **自然会正式接触陆沉**
2. 陆沉发现母亲病情线索
3. 有人认出了红焰真正来历的一部分

## 四、卷一结尾终点状态（拍板版）

### 陆沉
- 已拿下蓝鲸杯冠军
- 已正式完成从蓝星离职
- 已搬入基地 / 新生活雏形建立
- 已从"打工工具人"变成"有名字的赛手"
- 但仍然缺钱、缺体系、缺对更大世界的理解

### 红焰
- 完成卷一第一次重大成长
- 从"怪鱼"变成"圈内知名黑马核心鱼"
- 与陆沉建立稳定共斗关系
- 真实身份仍未完全公开，但危险感和特殊性已被更多人注意到

### 陈海
- 从"拽兄弟去钓鱼的人"升级成真正搭档
- 与陆沉关系更稳
- 与父亲旧线的关系可以在卷尾或卷二初揭开第一层

### 张海明 / 赵洋
- 阶段性败北
- 张海明现实线不一定彻底死，但卷一必须输得很难看
- 赵洋保留为"比赛线真正强敌"的余味

### 世界观
- 蓝鲸杯只是入口
- 龙门杯、自然会、斗王计划、父亲旧线、母亲病线都已经不是影子，而是下一卷真正要压上来的东西
