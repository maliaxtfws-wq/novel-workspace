# 卷一中期推进会纲（正式版）

> 版本：v1.0
> 定位：卷一第二阶段详细执行方案，可指导后续正文拆章
> 承接：WWB001-014"断线起势"之后，正式进入"草台班子立住"阶段

---

## 一、阶段总纲

### 阶段目标

把主角团从：

> 陆沉 + 陈海 + 红焰

推进成：

> 陆沉 + 陈海 + 周老板（半入队）+ 红焰 + 梭子 + 墩子

### 三件大事

1. **队伍结构立住**：一核两辅正式成型
2. **比赛认知立住**：陆沉第一次真正理解正规训练、声波指令、场地体系
3. **人物关系立住**：陈海升级、周老板升级、女主第一次亮相但不抢戏

### 阶段结束标志

- 梭子入队
- 墩子入队
- 第一版声波指令建立
- 第一次模拟对战翻车并完成复盘
- 蓝鲸杯入围赛赛程/抽签到来

## 三、第一小阶段：俱乐部开眼

### WWB015《后套间》

**核心内容**：
- 报名回来后的第一晚/第二天清晨
- 基地真正开始有生活感
- 后套间潮、乱、小，但两个人硬是收拾出"沉海战队临时指挥部"的感觉
- 陈海开始拍"创业vlog"
- 陆沉嘴上嫌弃，手上却已经开始认真规划红焰的喂养和训练时间

**章节作用**：把"搬进基地"从结果写成状态，建立中期日常底色。

---

### WWB016《周老板看不下去了》

**核心内容**：
- 周老板看两人瞎折腾红焰，连续指出问题
- 从喂养、换水、状态判断，到最基本的赛鱼照顾方式，全是漏洞
- 陆沉和陈海第一次意识到：他们连"养明白"都没做到，更别说"打比赛"

**章节作用**：正式推动周老板从"提供场地的人"升级成"专业前辈"。

---

### WWB017《俱乐部》

**核心内容**：
- 周老板带两人第一次进俱乐部
- 看见正式训练池、半职业队伍、赛鱼日常、基础指令训练
- 两人开眼：原来外面的世界已经卷成这样了
- 红焰单体依旧抢眼，但整体团队感极弱

**章节作用**：完成"俱乐部开眼"主事件，中期世界抬升第一步。

---

### WWB018《记录员》

**核心内容**：
- 女主第一次亮相
- 身份：俱乐部训练记录员/数据助理
- 不惊艳、不高调、不主动社交
- 但训练表、鱼只状态、记录流程，全是她默默收得最稳
- 陆沉第一次注意到她不是因为好看，而是因为她"像以前的自己"

**章节作用**：种下女主第一颗种子。

---

### WWB019《一盆冷水》

**核心内容**：
- 第一场俱乐部内模拟对抗/演示局
- 红焰强，但陆沉调度不行，陈海在场边更乱
- 没副手，红焰被迫一打多
- 被一支中等水平队伍教育
- 周老板一句话收尾：**"你们现在不是差一点，是差一整支队。"**

**章节作用**：用失败逼剧情进入"找队友鱼"阶段。

---

## 四、第二小阶段：河钓找鱼

### WWB020《回河边》

**核心内容**：
- 俱乐部受挫后，陆沉回到熟悉的河边
- 这次不是散心，是带着目的找鱼
- 他开始试着总结能力触发规律：鱼钩接触、反馈强弱、高价值鱼更明显
- 能力第一次有了"实验感"

**章节作用**：把陆沉从"偶然撞上天赋"推进到"开始有意识摸索天赋"。

---

### WWB021《不是只对红焰》

**核心内容**：
- 陆沉连续验鱼
- 普通鱼反馈弱，高价值鱼反馈明显
- 他终于确认：能力理论上不只对红焰有效，但不是每条鱼都值得触发
- 章节后半段埋入一条异常快、异常躁的目标鱼影

**章节作用**：中期关键章，能力边界第一次扩大。

---

### WWB022《梭子》

**核心内容**：
- 梭子正式上钩
- 漂相、速度、试探、抢口，全都很贼
- 陆沉差一点就错过
- 最终拿下第一条队友鱼：**紫品马口·梭子**

**章节作用**：正式完成"第一副手入队"。

---

### WWB023《不服》

**核心内容**：
- 梭子和红焰第一次见面
- 红焰看不起它
- 梭子不服，拼命炸刺
- 陆沉第一次意识到：**收一条鱼容易，让它进队难。**
- 周老板点评：梭子是把快刀，但太脆，扛不住

**章节作用**：把"得鱼"推进到"磨合"，自然过渡到"还缺盾位"。

---

## 五、第三小阶段：鱼市捡漏

### WWB024《鱼市》

**核心内容**：
- 陈海提出"去鱼市捡漏"的鬼点子
- 不是随口一说，而是提前踩过点、摸过门道
- 带陆沉去赛鱼市场
- 鱼市气氛热闹、混乱、吹牛、砍价，真假难辨

**章节作用**：把陈海往"战队外部运营核心"上推。

---

### WWB025《验鱼》

**核心内容**：
- 两人开始用极其土、极其骚的办法偷偷验鱼
- 陆沉负责看颜色反馈
- 陈海负责扯皮、砍价、转移注意力
- 这一章偏轻松、偏搞笑，是中期调味章

**章节作用**：把"超能力 + 市井感 + 兄弟喜剧感"揉在一起。

---

### WWB026《墩子》

**核心内容**：
- 墩子正式出场
- 第一眼看着丑、土、闷，不起眼
- 但反馈不对劲
- 最终以捡漏价拿下：**紫品土鲶·墩子**
- 周老板确认：这鱼不花哨，但是真能扛

**章节作用**：一核两辅结构正式闭环。

---

## 六、第四小阶段：草台班子成军

### WWB027《三条鱼》

**核心内容**：
- 红焰、梭子、墩子首次正式合训
- 结果不是战队，是事故现场
- 梭子乱冲，红焰嫌弃，墩子慢半拍
- 陆沉喊指令越喊越乱
- 陈海在旁边急得像教练又像保安

**章节作用**：让读者看到"鱼齐了不等于队成了"。

---

### WWB028《声波》

**核心内容**：
- 周老板开始系统教最基础的声波指令
- 声波指令是所有赛手的通用能力，差别在深度
- 陆沉的成长点：从乱喊到会少喊、会看时机、会留空当
- 红焰最先吃透，梭子最抗拒，墩子最慢热

**章节作用**：正式建立第一版战术语言。

---

### WWB029《翻车》

**核心内容**：
- 第一场真正像样的模拟对战
- 鱼都没问题，输在配合
- 梭子抢节奏、墩子替红焰吃伤、红焰想收割却被打断
- 陆沉临场判断迟半拍，整局崩掉

**章节作用**：中期最重要的一次"疼点"，这场输掉以后队伍才会真正成长。

---

### WWB030《复盘》

**核心内容**：
- 输完不散伙，主角团第一次像职业队一样复盘
- 陈海开始发挥真正价值：录像、记录、拉练习对象、做资料
- 女主可以在这里第二次小亮，默默补上某项关键记录或规则提醒
- 周老板收束战术定位：红焰主核、梭子机动、墩子护核

**章节作用**：把"翻车"变成"真正成长"。

---

### WWB031《赛程》

**核心内容**：
- 蓝鲸杯入围赛赛程/抽签/检录通知下来
- 队伍终于勉强有了样子
- 但大家心里都清楚：还没准备好
- 最后一幕落在基地深夜：三条鱼安静待着、陈海刷着赛程、陆沉盯着通知、谁都没说大话

**章节作用**：中期结束章，直接推进到"第三阶段：入围赛开打"。

## 七、章节命名风格说明

本版章节名延续已有习惯：

| 风格 | 示例 |
| ---- | ---- |
| 简短好记 | 《后套间》《俱乐部》《验鱼》《翻车》 |
| 有画面感 | 《梭子》《墩子》《声波》《赛程》 |
| 不装文学 | 《不服》《记录员》《复盘》 |
| 保留钩子感 | 《不是只对红焰》《一盆冷水》 |

与前面的《基地》《红焰》《英雄》《报名点》一脉相承。

## 九、梭子·墩子设定速查

### 梭子

| 项目 | 内容 |
| ---- | ---- |
| 品种 | 紫品马口 |
| 颜色 | 紫 |
| 定位 | 机动位/探路/扰乱/补刀 |
| 性格 | 小、快、躁，像不服输的小钢炮 |
| 特点 | 攻击性强、速度快，但太脆，站不住 |
| 与红焰关系 | 互相不服，红焰压制梭子，梭子拼命想证明自己 |

### 墩子

| 项目 | 内容 |
| ---- | ---- |
| 品种 | 紫品土鲶 |
| 颜色 | 紫 |
| 定位 | 抗压位/卡位/承伤/断后/护核 |
| 性格 | 稳、闷、皮实，像老黄牛 |
| 特点 | 不起眼，但颜色反馈不对劲；别人不要的，主角捡漏 |
| 与红焰关系 | 默默扛线，给红焰兜底 |

## 十、女主定位（中期亮相版）

| 项目 | 内容 |
| ---- | ---- |
| 身份 | 俱乐部数据助理/训练记录员 |
| 出场方式 | 不起眼、不多话、被人使唤、被默认忽略 |
| 核心特点 | 活做得又稳又准，一句废话没有 |
| 陆沉注意到她的原因 | 从她身上看见过去的自己——被低估但不说废话的人 |
| 中期定位 | 种子阶段，读者记住但不进团 |
| 后续原则 | 先亮相，再进团；必须符合剧情需要 |
