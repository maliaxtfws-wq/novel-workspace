# 卷一中期的纲

> 版本：v1.0
> 定位：卷一第二阶段（备赛立身）详细执行方案
> 承接：卷一会纲·阶段2"备赛立身"
> 当前正文进度：WWB013-014（离职+报名完成）

---

## 一、阶段定位

承接"断线起势"之后，正式进入第二阶段：**草台班子立住**。

这一段不是单纯过渡，而是整卷最重要的"立队期"——要完成基地、队友、训练、队伍成型这些关键任务。

核心是把主角团从：

> 两个刚离开旧生活的普通人 + 一条被低估的红焰

推进成：

> 一支虽然穷、土、草台，但已经有雏形、有打法、有奔头的黑马队伍

**气质要求**：轻松、搞笑、热血、奋斗。不是闲着玩鱼，而是一群各有难处的人，凑在一起狠狠干一把。

---

## 二、中期总目标

中期需要完成六件事：

1. 让"基地"真正活起来，有生活感、穷劲、草台气
2. 让周老板从工具人升级为前辈型核心配角
3. 让陈海价值升级，负责运营、资源、短视频、拉赞助等"战队外脑"工作
4. 完成两条过渡队友鱼入队：先**梭子**，再**墩子**
5. 建立队伍第一版战术体系：红焰主核、梭子机动、墩子抗压，一核两辅
6. 让陆沉第一次真正接触"职业边缘世界"，意识到自己离正式比赛还差得远

---

## 三、中期总矛盾

核心矛盾从"要不要辞职"转变为：**陆沉已经决定上赛道，但他和陈海根本不会跑。**

### 三层矛盾

**1. 现实层**
- 刚离职，钱紧
- 基地破破烂烂
- 没职业资源
- 主角团没人真正打过正规赛

**2. 队伍层**
- 只有红焰一个核心，阵容残缺
- 红焰强，但"一条鱼"打不了完整团队赛
- 梭子和墩子的加入不是数量补齐，而是战术位补齐

**3. 认知层**
- 陆沉的超能力当前只在第一阶段"鱼色"
- 能力只是杠杆，不是答案
- 他不是控制鱼，而是通过训练、声波指令、信息差判断和默契来影响比赛

---

## 四、中期四小阶段结构

### 第一小阶段：俱乐部开眼

**阶段任务**：让主角团第一次进入"半职业世界"，认识到自己到底有多菜。

#### 必须解决的问题
- 周老板为什么值得信
- 主角团为什么必须去俱乐部
- 陆沉为什么不能再靠"运气好+红焰猛"混下去
- 女主为什么能自然亮相但不抢戏

#### 建议剧情节点

**① 基地初体验**
- 报名回来后，写一小段基地生活
- 后套间逼仄、潮、乱，但被两人收拾出一点"要干事"的样子
- 陈海开始拿手机拍"沉海战队创业vlog"
- 陆沉嘴上嫌弃，实际上开始认真规划红焰喂养、换水、训练时间

**② 周老板看不下去**
- 周老板观察两人折腾红焰，指出一堆问题：
  - 喂法不对
  - 休息节奏不对
  - 训练全靠想当然
  - 连最基础的赛前检录习惯、伤病观察都不懂
- 不是骂人，而是老手看新兵蛋子犯蠢的无奈

**③ 带去俱乐部"开眼"**
- 周老板把两人带进熟识俱乐部
- 第一次看到正式训练池
- 第一次看到别人的赛鱼
- 第一次看到声波指令的雏形使用
- 第一次意识到：红焰很强，但他们这个团队很土

**④ 女主首次亮相**
- 身份：俱乐部数据助理 / 训练记录员
- 感觉：不起眼、不多话、被人使唤、被默认忽略
- 但所有训练数据、登记、记录、鱼只状态，她都做得最细
- 陆沉第一次注意到她不是因为好看，而是：
  > 她被人低估、被当工具用，但她一句废话没有，活做得又稳又准。陆沉从她身上，看见了过去的自己。

**⑤ 第一轮打击**
- 安排一场训练演示或模拟对抗
- 红焰单体表现依旧亮眼
- 但整体调度一塌糊涂
- 陈海乱喊，陆沉指令滞后
- 没有副手，红焰被迫一打多
- 被俱乐部里中等水平队伍教育

**阶段结尾钩子**

> 陆沉彻底意识到：他们现在不是差一点，是差一整支队。
> 周老板给出关键判断：**"你们先别想着怎么赢，先想想你们还缺谁。"**

---

### 第二小阶段：河钓找鱼

**阶段任务**：验证陆沉能力边界，拿下第一条队友鱼**梭子**。

#### 核心价值
- 验证能力边界：陆沉的能力不是只对红焰有效
- 让红焰第一次产生"领地意识/排斥心理"
- 让队伍第一次出现"磨合问题"
- 红焰不是工具鱼，它有性格

#### 建议剧情节点

**① 回到野河**
- 俱乐部受挫后，陆沉不服，回到熟悉的河边
- 这次不是散心，而是带着目的去找鱼

**② 陆沉第一次主动"验鱼"**
- 用小钩试探
- 感知不同鱼的颜色反馈
- 逐渐确认高价值鱼反馈更强，普通鱼反馈很弱
- 写出"能力开始从偶然变成方法"

**③ 梭子登场**
- 定位：紫品马口，小、快、躁，像个不服输的小钢炮
- 上钩方式：漂相贼、速度快、几次试探、突然接口/黑漂/走漂
- 陆沉反应稍慢就会跑
- 这种鱼适合把"技术感"写出来

**④ 红焰排斥梭子**
- 红焰不服，梭子也不服
- 一个是天才王者，一个是小炮仗
- 一见面就互相看不顺眼
- 写出"强者对新人的天然压制"和"普通战友拼命想证明自己"的关系

**⑤ 陆沉第一次意识到"队伍比单核难太多"**
- 两条鱼节奏不同、性格不同、指令逻辑不同
- 红焰未必愿意带新人
- 从"单鱼羁绊"升级到"队伍调度"

**阶段结尾钩子**

> 训练里，梭子虽然快，但太脆，根本站不住。周老板一句话：**"你们现在有刀了，但还没有盾。"**

---

### 第三小阶段：鱼市捡漏

**阶段任务**：陈海主导，用"验鱼"鬼点子去鱼市捡漏，拿下**墩子**。

#### 陈海价值升级
- 不是灵光一闪，而是提前踩点、做准备
- 知道哪些鱼表面不值钱、实际可能有潜力
- 想好怎么偷拍视频、怎么做短视频素材
- 运营脑子出来

#### 建议剧情节点

**① 陈海拿出方案**
- 提前踩鱼市
- 研究摊主套路
- 想好怎么给"沉海战队"攒第一波曝光

**② 鱼市段落要写得轻松好玩**
- 摊主吹牛
- 陈海贫嘴砍价
- 陆沉表面淡定，实际偷偷验鱼
- 像逛古玩市场捡漏
- "我要验鱼"这个梗第一次落地

**③ 墩子登场**
- 定位：紫品土鲶，稳、闷、皮实、能扛
- 角色：卡位/承伤/断后/护核位，像老黄牛
- 出场方式：丑、土、不起眼、看着有点笨，但颜色反馈不对劲
- 典型的"别人不要、主角捡漏"

**④ 墩子的价值不是战斗帅，是"让队伍第一次像队伍"**
- 红焰负责主杀
- 梭子负责探路、扰乱、补刀
- 墩子负责扛线、卡位、兜底
- 主角团第一次能打出"配合"的雏形

**阶段结尾钩子**

> 鱼齐了，但一上手训练发现：三条鱼放在一起，不是战队，是事故现场。

---

### 第四小阶段：草台班子成军

**阶段任务**：完成真正意义上的队伍磨合、第一版声波指令系统建立、第一次模拟对战翻车再进步，直到赛程/抽签到来。

这是整个中期最重要的一段，决定读者会不会相信这支队伍能进比赛。

#### 建议剧情节点

**① 训练蒙太奇开始**
高节奏连续推进：
- 基地训练
- 俱乐部借场
- 喂养调整
- 伤病观察
- 配合测试
- 失败复盘
- 再试一次

重点写出"土办法 + 笨办法 + 死磕"的成长感。

**② 声波指令第一版建立**
- 最开始喊啥都乱
- 后来慢慢学会少喊
- 学会分层下令
- 学会"给红焰判断空间"
- 学会"让梭子去探、让墩子去扛"
- 陆沉从"发号施令"成长为"真正指挥"

> 核心原则：声波指令是所有赛手的通用能力，陆沉的优势是判断窗口，不是独占指挥权。

**③ 三条鱼的关系跑顺**
- 红焰一开始看不起梭子和墩子
- 梭子不服，老想证明自己
- 墩子话最少，但最能扛
- 慢慢地，红焰开始默认他们存在
- 梭子开始信红焰
- 墩子开始给红焰让位、兜底

**④ 第一次模拟对战翻车**
- 不能是"打不过别人"，而要是"明明鱼没问题，输在配合"
- 梭子抢节奏过头
- 红焰节奏被打乱
- 墩子为了护核提前吃伤
- 陆沉临场判断迟半拍
- 陈海在场边急得乱吼，反而干扰判断
- 输完复盘，不能灰

**⑤ 陈海真正进入"外脑位"**
- 录像复盘
- 剪训练片
- 记对手特征
- 想办法找练习对象
- 补没有的钱买不到的东西
- 形成：陆沉=场内脑，陈海=场外脑

**⑥ 女主第二次小亮**
- 默默把训练记录整理好
- 提醒某项检录规则
- 指出某次失败不是鱼的问题，而是调度节奏问题
- 说完继续做事，不邀功

**⑦ 抽签/赛程到来（中期收束点）**
- 队伍有了第一版成型打法
- 三条鱼勉强能打
- 大家还没完全稳
- 赛程却已经来了

---

## 五、中期结束标志（四条）

1. **队伍结构稳定**：红焰、梭子、墩子三鱼定位明确
2. **主角团关系稳定**：陆沉负责指挥与判断，陈海负责外部运营与资源统筹，周老板负责专业训练与规则兜底
3. **女主完成"第一次有效种子"**：读者记住她，但她还没正式进主角团
4. **入围赛前夜氛围立住**：草台班子终于有了样子，但仍然穷、土、险、没底气

## 六、梭子、墩子设定速查

### 梭子

| 项目 | 内容 |
| ---- | ---- |
| 品种 | 紫品马口 |
| 颜色 | 紫 |
| 定位 | 机动位/探路/扰乱/补刀 |
| 性格 | 小、快、躁，像个不服输的小钢炮 |
| 特点 | 攻击性强，速度快，但太脆，站不住 |
| 与红焰关系 | 互相不服，红焰压制梭子，梭子拼命想证明自己 |

### 墩子

| 项目 | 内容 |
| ---- | ---- |
| 品种 | 紫品土鲶 |
| 颜色 | 紫 |
| 定位 | 抗压位/卡位/承伤/断后/护核 |
| 性格 | 稳、闷、皮实，像老黄牛 |
| 特点 | 不起眼，但颜色反馈不对劲；别人不要的，主角捡漏 |
| 与红焰关系 | 默默扛线，给红焰兜底 |

## 八、气质控制（防两个问题）

### 必须防止写成纯设定说明书
- 要靠训练翻车、鱼之间的脾气、主角团穷酸日常、陈海贫嘴、周老板嘴硬心软来把设定"演出来"

### 必须防止写成无脑升级流
- 陆沉刚开始真不懂、真会输、真会出丑、真得一遍一遍练
- 只有这样，后面入围赛开打，读者才会觉得爽

---

## 九、中期一句话总结

> 离开公司以后，陆沉和陈海没能立刻变强，他们只是先学会了，怎么把一支破破烂烂的队伍，硬撑成一支能上场的队伍。
