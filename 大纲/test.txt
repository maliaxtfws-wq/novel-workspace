## 场景一：早上 · 浑浑噩噩

**地点**：出租屋 → 地铁 → 公司工位

| 节点 | 内容 |
|------|------|
| 起床 | 7:00闹钟响，一夜没睡好，脑子里翻来覆去是陈海那句"我们是兄弟" |
| 出门前 | 照镜子，黑眼圈比昨天更重 |
| 地铁上 | 刷手机，看到陈海的朋友圈——钓鱼的短视频，配文"今天是个好日子" |
| 心理 | 陆沉点了个赞，又取消了，然后又点了一个 |
| 到公司 | 坐在工位，打开电脑，盯着代码发呆 |
| 状态 | 眼睛看着屏幕，脑子里全是问号——"我昨晚是不是做错了？" |

**陆沉的不安（第一层）**：陈海的事
- 昨晚那条"我们是兄弟"，他没回
- 十几个未接来电，他也没接
- 他不知道怎么开口说"对不起"

**陆沉的不安（第二层）**：对现实的不安
- 凌晨两点还在改bug，早上七点就要起床
- 干了四年，还是T2-2，还是"高级工程师"
- 晋升公示又扫了一眼，没有他
- 张海明昨天那句话："别分心"
- 什么叫"别分心"？是警告吗？

**陆沉的自嘲**："我这个人，记性不好。"
- 反复对自己说
- 其实不是记性不好，是不知道怎么面对

## 场景二：上午-下午 · 低效率的一天

**地点**：公司工位

| 节点 | 内容 |
|------|------|
| 9:00-12:00 | 写代码，但频繁走神，bug比平时多 |
| 午饭 | 食堂一个人坐，看着手机通讯录里陈海的头像 |
| 13:00-15:00 | 开会，全程心不在焉，被张海明点名问问题，答非所问 |
| 张海明反应 | 皱眉，没说什么，但那个眼神让陆沉心里咯噔一下 |
| 15:00 | 刘健路过工位，看到陆沉发呆 |

**15:00刘健对话（修复后）**：

> 刘健："沉哥，你今天怎么了？魂不守舍的。"
>
> 陆沉："没睡好。"
>
> 刘健："是因为陈海的事吧？"
>
> 陆沉愣了一下："……你怎么知道？"
>
> 刘健："今天中午在食堂碰到他了，他说你昨天临时有事，下次再约。他还说，陆沉这人就是这样，经常有事，你放心，他肯定来。"
>
> 陆沉："……他还说什么了？"
>
> 刘健："没了。就这些。不过我看他的样子，好像有点失落。"
>
> 陆沉沉默了一会儿："……谢了。"
>
> 刘健拍拍他的肩膀："没事。你们俩那么铁，有什么说不开的？"

**陆沉心理**：
- 陈海没有在外面乱说，只是说"临时有事"
- 但那句"有点失落"，扎到他了
- 他想起昨晚那些没接的电话

**陆沉的不安（第三层）**：对前途的思考
- 干到三十岁怎么办？
- 继续修bug？继续被张海明骂？
- 50万，蓝鲸杯，真的能成吗？
- 他一个连金鱼和锦鲤都分不清的人，去参加斗鱼比赛？

**自嘲**："天赋在我身上的体现就是——完全没有天赋。"
- 他对自己说
