## 一、鱼灵体系澄清（核心定义修订）

### 旧理解（错误）
鱼灵颜色 = 这条鱼的强弱

### 新理解（正确）
```
赛事评级 = 这条鱼"现在"有多强（可量化，客观）
鱼灵颜色 = 这条鱼灵魂的强度 = 成长上限（天生注定）
```

**类比**：宝可梦的个体值 vs 种族值
- 两条同物种小火龙，种族值相同，个体值不同
- 个体值31的那只培养到满级时比个体值15的强很多
- **物种基础分** = 种族值（赛事评级）
- **鱼灵颜色** = 个体值（灵魂强度）

### 放射性矿物的真正作用
- 强化生物本能和基因潜能
- 最关键：增强灵魂强度（成长性）
- 鱼类受影响后可能突破物种上限
- 人类受影响后可能获得"感知鱼灵"的能力

### 陆沉超能力的来源
```
不是天赋异禀，而是：
1. 接触了被矿物影响的水源/鱼（父亲渔具店相关？）
2. 自身基因在矿物作用下发生变异
3. 灵魂强度因此超过普通人
4. 接触鱼竿时能"感应到"鱼的灵魂（像调频收音机）
```

## 二、蓝鲸杯入围赛规则（深渊大逃杀）

### 核心定位
128人同时入场，混战30分钟，生存博弈而非单挑

### 场地
T03「深海暗沟」—— 完全黑暗，视野3米，有回声定位

### 晋级规则
| 项目 | 数据 |
|------|------|
| 参赛人数 | 128人 |
| 每人参赛鱼 | 6条 |
| 总鱼数 | 768条 |
| 晋级名额 | 32人（25%） |
| 晋级门槛 | 500积分 |
| 战斗时长 | 30分钟 |

### 积分规则

| 行为 | 积分 |
|------|------|
| 击杀A级鱼（最后一击） | 100分 |
| 击杀B级鱼 | 50分 |
| 击杀C级鱼 | 20分 |
| 击杀D级鱼 | 10分 |
| 自己的鱼存活 | 该鱼评分分值 |

### 晋级门槛演算

**典型选手得分推算：**

| 选手类型 | 击杀得分 | 存活得分 | 总计 | 结果 |
|---------|---------|---------|------|------|
| 普通人（鱼全灭，击败5条） | 100分 | 0分 | 150分 | ❌淘汰 |
| 中上水平（鱼全灭，击败12条） | 300分 | 0分 | 300分 | ❌淘汰 |
| 前几名（存活3条，击败15条） | 390分 | 170分 | 560分 | ✅晋级 |

**结论**：500分门槛合理，符合正态分布，约32-35人能晋级

## 三、赛事方评级体系（战鱼评分制）

### 三项积分维度

| 维度 | 评估内容 | 分值 |
|------|---------|------|
| 物种基础分 | 鱼种分A/B/C/D四级 | 0-40分 |
| 技能加分 | 已认证技能的稀有度和强度 | 0-30分 |
| 战绩加分 | 过往比赛记录（胜率×场次） | 0-30分 |
| **总分** | 三项加总 | 0-100分 |

### 物种基础分（A/B/C/D四级）

| 等级 | 代表鱼种 | 基础分 |
|------|---------|--------|
| A级 | 龙鱼（银龙/青龙）、血鹦鹉、巨暹罗鲤 | 40分 |
| B级 | 改良罗非、地图鱼、鹦鹉 | 28分 |
| C级 | 普通鲫鱼、草鱼、鳜鱼 | 18分 |
| D级 | 小型观赏鱼（金鱼、孔雀鱼） | 8分 |

### 技能加分

| 技能类型 | 加分 |
|---------|------|
| 普通攻击技/防御技 | +3分/个 |
| 稀用攻击技/防御技 | +5分/个 |
| 控制技 | +6分/个 |
| 被动技 | +4分/个 |

### 红焰的赛事评分（估算）

| 维度 | 数据 | 得分 |
|------|------|------|
| 物种基础分 | A级（巨暹罗鲤） | 40分 |
| 技能加分 | 「飞跃」+5（待认证） | 5分 |
| 战绩加分 | 0（白纸鱼） | 0分 |
| **总分** | — | **45分（B+级）** |

> 赛事方评估：B+级鱼。实际鱼灵：红色（成长上限远超B+）。

## 四、入围赛「深渊大逃杀」四阶段策略

### 阶段一（0-10分钟）：潜伏期
- 红焰钻入最深的暗沟角落
- 释放低频声波探测周围鱼群分布（不暴露位置）
- 找岩石缝作临时巢穴
- 目标：不动、不打、不露面，观察所有鱼的分布和品级

### 阶段二（10-20分钟）：搅局期
- 陆沉通过超能力感知密集区域
- 红焰在暗处游走，故意从大群鱼边缘掠过
- 制造动静引诱区域内鱼互相攻击
- 目标：让其他队伍减员，红焰保持100%体力

### 阶段三（20-25分钟）：收割期
- 场上约40-50条鱼，大部分筋疲力尽
- 红焰体力仍充沛，选定"正在内讧"的区域突袭
- 趁乱击杀2-3条体力耗尽的鱼，积累积分

### 阶段四（25-30分钟）：决赛圈
- 剩10-15条鱼，局面稳定成"小势力对峙"
- 红焰是场上体力最充沛的红灵鱼
- 陆沉选择最后对手，主动出击
- 目标：拿下决定性击杀，确保总分排名

## 五、灵魂赏金规则（待战中摸索）

### 核心机制
击杀"鱼灵颜色远高于赛事评分"的鱼，可获得额外灵魂赏金

| 鱼灵颜色 | 灵魂赏金 |
|---------|---------|
| 蓝灵 | +30分 |
| 紫灵 | +50分 |
| 金灵 | +80分 |
| 红灵 | +120分 |

### 灵魂赏金来源
不是赛事方规则，是陆沉在反复比赛中**自己摸索出来的规律**

### 信息差逻辑
- 陆沉知道：去打红灵鱼，积分收益最高（风险也最大）
- 陆沉知道：有些"被低估的隐藏强者"要躲开
- 赛事方/对手：完全不知道这套逻辑
