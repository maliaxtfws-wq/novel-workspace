# 斗鱼竞技场地设定

> 2026-03-30 新增

---

## 场地规格

所有正式斗鱼比赛，举办单位必须提供统一规格的战斗场地：

| 项目 | 规格 |
|------|------|
| **面积** | 标准足球场大小 |
| **高度** | 约五层楼高（≈15米） |
| **形状** | 立方体 |
| **单场容量** | 约100+条鱼同时搏斗 |

---

## 场地主题（48个）

场地内壁与底部根据主题不同，呈现完全不同的生态环境。选手需要根据场地主题制定战术。

### 代表性主题

| 主题 | 环境特征 |
|------|----------|
| **亚马逊丛林** | 底部藏有水草、树木、岩石，地形复杂 |
| **海洋/深海** | 开阔水域，暗流，视野开阔 |
| **火山表面** | 底部有熔岩纹路、礁石，温差暗流 |
| **亚马逊河流** | 湍急水流，水草丰茂，迂回空间大 |
| **沉船遗迹** | 底部有旧船骸、人工礁石，可藏鱼 |
| **红树林** | 根系发达，遮蔽物多，适合伏击型鱼种 |
| **珊瑚礁** | 地形错落，缝隙多，利于防守反击 |
| **城市排水系统** | 工业风，管道穿梭，高速通道 |

> 完整48个主题已具体化如下。

## 战术维度

### 地形利用

复杂地形（亚马逊丛林、红树林、珊瑚礁等）提供的战术选项：

- **躲藏伏击**：利用水草/岩石掩护，诱敌深入后突袭
- **迂回包抄**：借树木/根系遮挡，侧翼进攻
- **环境压制**：利用暗流、温差等自然条件消耗对手

开阔地形（海洋、火山表面等）则更依赖：

- **正面冲击**：速度和力量的直接碰撞
- **阵型拉扯**：团队配合分割对手

### 单场规则

| 项目 | 规则 |
|------|------|
| **参赛鱼类上限** | 约100+条/场 |
| **选手出场鱼数** | 每人最多6条 |
| **单场胜负** | 规定时间内，存活鱼多者胜 |

## 鱼类技能系统

每条鱼最多拥有 **4个技能**。

### 技能来源

- **天生技能**：鱼类品种决定，的血统差异
- **训练解锁**：通过后天训练激发
- **协同强化**：同队鱼类技能产生联动效果

### 技能类型（示例）

| 类型 | 效果 |
|------|------|
| 攻击技 | 直接伤害、穿透 |
| 防御技 | 减伤、护盾 |
| 控制技 | 冰冻、缠绕、致盲 |
| 机动技 | 加速、瞬移、闪现 |
| 辅助技 | 治疗、增益队友 |
| 被动技 | 永久生效的特性强化 |

### 团队协作

技能之间的配合是核心战术深度：

- **技能连携**：A鱼技能为B鱼创造输出窗口
- **阵型buff**：特定鱼种组合激活额外加成
- **诱饵战术**：用低价值鱼引诱对手暴露技能CD

---

## 战略深度

**场地主题 × 鱼类技能 × 团队配置**

三个维度的排列组合形成了斗鱼竞技的策略深度：

```
优秀选手 = 了解自己的鱼 + 熟悉场地主题 + 能临场应变
顶级选手 = 上述 + 能通过训练提前适配 + 能打出预判外的配合
```
