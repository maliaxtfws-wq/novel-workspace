## 一、文档定位

这份文档不是完整人物卡合集，而是**俱乐部线功能生态表**。

核心目标：

> **让"俱乐部"不只是一个训练场景，而是一个有层级、有秩序、有行业气味，能持续产出剧情冲突的空间。**

---

## 二、俱乐部整体定位

### 俱乐部名称

**浪潮斗鱼俱乐部**

> 低调务实路线，不搞中二，不做豪门感。它不是顶级豪门，而是"半职业边缘、在本地圈子里有点东西"的那种地方。

### 俱乐部层级定位

- 不属于全国顶尖
- 但在浅圳本地算有门路、有训练体系、有老选手资源
- 对普通民间选手来说，已经是"专业世界"
- 对真正顶级队伍来说，又还差一层

这个层级最适合中期，因为：
- 能让陆沉和陈海"开眼"
- 又不会把天花板一次性掀太高
- 保留后面更大世界继续抬升的空间

### 俱乐部气质

> 带点旧水族圈味道，带点半职业训练馆气息，带点现实生存压力的地方。

关键词：
- 规矩多
- 人情也多
- 真有人懂行
- 也真有人混日子
- 外人看热闹，圈里人看门道

## 四、核心层角色

### 周老板

详见 `人物/周老板.md`

在俱乐部生态里的作用：
1. **关系入口**：没有周老板，主角团进不来
2. **专业翻译器**：把俱乐部里专业内容"翻译"给主角团
3. **信任担保**：让俱乐部老板愿意先看看陆沉这帮人
4. **旧圈子锚点**：让俱乐部不只是新世界，也是"父亲那代人残影还在"的地方

| 项目 | 内容 |
| ---- | ---- |
| 姓名 | 顾启明 |
| 年龄 | 四十岁上下 |
| 定位 | 浪潮斗鱼俱乐部老板 |
| 性格 | 说话不重，但话里都带算盘；不恶毒，不反派；标准"这个圈子不是慈善堂"的现实派 |

1. 现实压力制造者：永远优先考虑场地成本、训练资源、鱼损耗、谁值不值得培养

**典型台词**：
- "热血我见得多，能留下来的没几个。"
- "鱼好，不代表队伍就成。"
- "训练池不是不要钱的，你们先拿点像样的东西给我看。"

**是否要正式人物卡**：先不做，等他后面真的进入主线资源线再升正式卡。

| ---- | ---- |
| 姓名 | 许青禾 |
| 定位 | 俱乐部数据助理/训练记录员 |
| 出场方式 | 不起眼、不多话、被人使唤、被默认忽略 |
| 核心特点 | 活做得又稳又准，一句废话没有 |
| 陆沉注意到她的原因 | 从她身上看见过去的自己——被低估但不说废话的人 |

**在俱乐部生态里的作用**：
1. 自然亮相点：不靠"惊艳"，靠"做事稳"
2. 后勤秩序核心：训练记录、鱼只状态、设备登记、排表、检录流程
3. 行业真实感承载者：代表俱乐部里最容易被忽视、但其实最离不开的人
4. 后续入团种子：先亮相，再进团，不抢中期主线

**使用原则**：
- 前期戏份要克制
- 不要一亮相就给她过多心理戏
- 最有效的写法：别人都在说，她已经把事做完了

## 七、俱乐部内部关系网

### 关系结构（简版）

```
顾启明（老板）——掌控资源与场地，现实派
       ↑
周老板（顾问）——懂行、有人情面，带路人
       ↑
唐振（陪练队长）——训练标准线
       ↑
魏小川/孙浩（陪练队员）——执行层
       ↑
许青禾（记录员）——秩序与细节支撑（谁都觉得她理所当然）
       ↑
梁叔（老训练师）——老圈子观察者
       ↑
何苗（检录执行）——流程与制度外壳
       ↑
阿杰（学徒）——生活感与场景活气
```

### 关系逻辑

1. **顾启明 ↔ 周老板**：互相认识多年，有信任但也不是毫无分歧。一个偏现实，一个偏看人。
2. **周老板 ↔ 梁叔**：老熟人，适合承担"老一辈看门道"的闲聊段落。
3. **许青禾 ↔ 俱乐部众人**：谁都默认她在，但很少有人真正重视她。正好给主角团"看见她"留空间。
4. **唐振 ↔ 顾启明**：顾启明信他稳，所以他很适合当训练试金石。

## 九、使用原则

1. **俱乐部不是新主角团**：配角再多，也只能服务主角团成长。不要让俱乐部人物自己长篇大论抢戏。
2. **一次只推一个新重点**：WWB017重点是"俱乐部世界"，WWB018重点是"女主第一次被看见"，WWB019重点是"被教育+看见差距"。不要一章塞满所有功能人物。
3. **功能位先于完整人物弧线**：陪练队员、检录、学徒，现阶段先保证好用。真写出手感了，再决定要不要升正式人物卡。
4. **每个人都要有"职业味"**：就算是功能位，也要像是在这个地方长期待着的人。不能像为剧情临时站出来的纸片人。
