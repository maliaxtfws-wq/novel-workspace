# 卷一场地详设

> 版本：v1.0
> 用途：服务卷一中期训练线、入围赛、正赛前半段的场地表现与战术设计
> 定位：卷一级执行文件 / 场地表现层
> 底层参照：`斗鱼竞技规则总纲.md`、`斗鱼竞技场地设定.md`
> 更新时间：2026-04-05

## 一、使用原则

### 1. 场地不是背景板，是半个对手

每个场地都必须能"主动影响战斗"，不是换个皮肤，而是换一种博弈。

### 2. 每个场地只突出 1—2 个核心机制

读者记住的永远应该是：
- 这张图为什么难打
- 这张图最怕什么
- 这张图谁容易吃亏

### 3. 同一支队伍，在不同场地必须像不同版本

这才会产生"懂地形的人"和"只会硬碰硬的人"的差距。

### 4. 俱乐部训练必须模拟真实赛场

从 WWB017 开始，训练就不能只是"标准池里喊两嗓子"。

**正确方向**：

> 俱乐部拥有模块化训练池，可以模拟正式赛场的地形、造流、光照、遮蔽物和局部规则。

训练本质是：先在俱乐部"见地图" → 再在模拟图里挨打 → 再在特定图里练针对战术 → 最后到正式比赛里兑现。

## 二、俱乐部训练系统

### 浪潮斗鱼俱乐部训练池结构

俱乐部拥有"半比例战术模拟池"，分为三层：

| 层级 | 用途 | 功能 |
| ---- | ---- | ---- |
| A. 基础池 | 喂养，日常观察，基础指令训练 | 看状态，练响应，练单鱼动作 |
| B. 模块池 | 单一规则强化训练 | 只模拟一种核心机制 |
| C. 对抗池 | 半比例实战模拟 | 多种模块组合成"拟真赛场" |

### 训练正确推进方式

**第一阶段**：先学会"看地图"——不懂地形的人，带再好的鱼进场也都是在送。

**第二阶段**：每张图只练一个重点。

**第三阶段**：训练赛要"带着地图目标"，不能每次都写成"打一场"。

**第四阶段**：正式比赛前，主角团已经"吃过地图的苦"。

## 四、八大场地详细设定

---

### T33 标准训练池

**一句话概括**：没有花活的地方，最能看出基本功。

**环境规则**：
1. 无障碍、无遮挡、无特殊增益区
2. 水流稳定，几乎没有随机变量
3. 所有队伍默认高能见度
4. 任何失误都会被直接放大

**优势鱼种**：正面硬碰硬型 / 基础体格强的鱼 / 指令响应高的鱼

**劣势鱼种**：依赖伏击的小型鱼 / 需要借地形拖时间的防守鱼

**红焰适配度**：8/10——正面能力强，但隐匿价值被削弱

**赵洋体系适配度**：5/10——标准池太空，辅助鱼不容易把战场拆开

**观感关键词**：空

---

### T09 沉船遗迹

**一句话概括**：看起来像废墟，实际上是会吃人的立体迷宫。

**环境规则**：
1. 船体内外存在多条窄道与穿插路线
2. 视野被残骸频繁切断，容易丢失目标
3. 某些区域只能容纳单鱼通过，容易形成"单点截杀"
4. 中央残骸区有高风险高收益位置，利于埋伏与反埋伏

**优势鱼种**：会绕障碍的机动鱼 / 适合伏击和突然切出的鱼

**劣势鱼种**：体型太大、转身慢的鱼 / 只会直线冲锋的队伍

**红焰适配度**：9/10——能借残骸翻线、绕出视野死角

**赵洋体系适配度**：7/10——能切割，但容错更低

**观感关键词**：藏

---

### T05 红树林根城

**一句话概括**：一座由根须构成的低矮森林，谁急谁先死。

**环境规则**：
1. 根系区域大量切碎直线路径
2. 部分通道只能中小型鱼穿梭，大鱼受限
3. 某些根区会减速，某些根间空道反而是高速突入线
4. 视野被反复切断

**优势鱼种**：灵活鱼、小快鱼 / 擅长边打边撤的骚扰型

**劣势鱼种**：纯正面压制型 / 体型大、转向慢的鱼

**红焰适配度**：7.5/10——能靠飞跃跨根、靠隐匿玩突然消失，但长期在狭窄根区不如梭子舒服

**赵洋体系适配度**：8/10——切割思路天然有利

**观感关键词**：绕

---

### T34 环形赛道

**一句话概括**：它不是地图，它是一条会逼着你不断做选择的跑道。

**环境规则**：
1. 沿主赛道移动会获得更流畅的加速收益
2. 抢内道拥有路线优势，但容错更低
3. 离开主赛道切中央，需要承担失速与暴露风险
4. 被对手黏住尾线时会持续承受追击压力

**优势鱼种**：速度型 / 节奏型 / 会追击、不怕持续换位的鱼

**劣势鱼种**：起步慢、转向慢的鱼 / 耐力差、容易被拖死的鱼

**红焰适配度**：8/10——爆发和飞跃可在关键点抢线，但真正吃红利的是梭子

**赵洋体系适配度**：4/10——怕被拖着跑，巨魶机动性差

**观感关键词**：快

---

### T17 火山表面

**一句话概括**：看似开阔，实际上整张图都在悄悄烧你。

**环境规则**：
1. 存在冷区、暖区与危险热障区
2. 部分热流会周期性改变路线
3. 长时间停留在高热区会持续消耗状态
4. 场地会"驱赶站桩者"，不允许队伍舒服抱团不动

**优势鱼种**：耐压、耐消耗、会换区的鱼 / 擅长拉扯区域资源的队伍

**劣势鱼种**：脆皮快鱼 / 需要长时间蹲点的鱼

**红焰适配度**：8.5/10——敢冲敢顶，适合高风险区域争夺

**赵洋体系适配度**：6/10——怕长时间被环境消耗

**观感关键词**：烧

---

### T23 暗河洞穴

**一句话概括**：当你看不见的时候，指挥本身才开始值钱。

**环境规则**：
1. 整体低能见度，局部几乎全黑
2. 洞壁与转角会制造视野盲区
3. 声波与水流反馈变复杂，容易误判位置
4. 长直冲刺空间少，多为短距离拐角接敌

**优势鱼种**：稳定型 / 敢贴墙走位的鱼 / 纪律性强的团队

**劣势鱼种**：纯视觉追击型 / 场边指挥混乱的队伍

**红焰适配度**：7/10——正面压迫会被环境削掉一部分，更依赖指挥和梭子探路

**赵洋体系适配度**：8.5/10——低信息图天然适合切割与伏击

**观感关键词**：盲

---

### T38 双层结构

**一句话概括**：上层是遮蔽，下层是修罗场，谁什么时候换层，决定谁死。

**环境规则**：
1. 上层遮蔽多，适合藏、绕，保核心
2. 下层相对开阔，适合正面接敌
3. 上下层连接口有限，容易形成抢门与断门
4. 上层安全但输出效率低，下层危险但收益高
5. 团队换层时最脆弱

**优势鱼种**：懂位置管理的团队 / 擅长上下层切换节奏的指挥者

**劣势鱼种**：只会抱团冲 / 不会分层执行的队伍

**红焰适配度**：9/10——能做最可怕的跨层收割者

**赵洋体系适配度**：7.5/10——要求极高换层判断，失误会被楼层牵着走

**观感关键词**：换

---

### T15 古井深渊

**一句话概括**：边缘安全，中心致命；但真正能赢的人，最后都得下井。

**环境规则**：
1. 边缘区相对安全，中心区高风险高收益
2. 井壁凸台提供短暂停位点，但容易被断退路
3. 越接近中心，压迫感越强，撤退成本越高
4. 时间拖久后，中心争夺不可避免

**优势鱼种**：敢打中心争夺的鱼 / 爆发强、胆子大的队伍

**劣势鱼种**：只会边缘磨蹭的队伍 / 心态不稳的新手

**红焰适配度**：9.5/10——飞跃、背水、压迫感在古井图会被放到最大

**赵洋体系适配度**：6.5/10——怕持续高压争中心

**观感关键词**：压

## 七、写作硬提醒

### 1. 场地一定要"先用，再解释"

正确顺序：先让人物在图里吃亏 → 再让周老板/唐振点出关键机制 → 这样读者才记得住。

### 2. 每张图都要有"观感关键词"

记住这个字，整张图就不会散：
- 沉船遗迹：藏
- 红树林根城：绕
- 环形赛道：快
- 火山表面：烧
- 暗河洞穴：盲
- 双层结构：换
- 古井深渊：压

### 3. 卷一不要贪多

卷一不是要把48张图全写完，而是要把 6—8 张图写得让读者记得住、期待下次再见。
