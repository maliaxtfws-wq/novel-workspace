# 斗鱼竞技规则总纲 v2.0

> 版本：v2.0
> 定位：斗鱼竞技体系唯一官方规则源文件
> 用途：统一竞技规则、评级体系、赛事路径、准入机制、反刷分逻辑
> 适用范围：蓝鲸杯、龙门杯、市级赛、赛区赛、全国赛、斗王挑战全链路
> 说明：若其他文件与本文件冲突，以本文件为准

---

## 第一部分：竞技定位

斗鱼竞技，是一种以**战斗型观赏鱼**为核心单位、以**场地规则 × 鱼类能力 × 团队指挥**为核心博弈维度的水族对抗竞技运动。

它不是简单的"谁的鱼更凶"，也不是单纯的"谁鱼种更贵"。
真正的竞技深度来自三个维度的叠加：

```text
场地主题 × 鱼类能力 × 团队指挥 = 战略深度
```

这意味着：

- 同一条鱼，在不同场地中的表现可能完全不同
- 同一支团队，在不同指挥风格下胜率可能天差地别
- 低评分战鱼并非毫无机会，只要信息差、地形利用、鱼种配合成立，依然可能以弱胜强

> **核心卖点**：
> 一个普通人，只要懂鱼、懂地形、懂节奏，也能击败拥有高评分战鱼却不会指挥的对手。

## 第二部分：三套判定体系（必须区分）

斗鱼竞技中，鱼的强弱不能只用一个指标来概括。
本体系采用三套彼此独立但互相关联的判定逻辑：

---

### 一、鱼灵颜色（世界观内部体系）

鱼灵颜色代表这条鱼的**灵魂强度与成长上限**。
它反映的是"潜力"，不是"当前公开战力"。

#### 颜色等级

| 鱼灵颜色 | 含义 | 潜力等级 |
| --------- | ---- | -------- |
| 白 | 极低 | 无成长性 |
| 绿 | 低 | 体格略优于普通鱼 |
| 蓝 | 中 | 可进入基础竞技层 |
| 紫 | 中高 | 已具备灵智雏形 |
| 金 | 高 | 稀有且成长性强 |
| 红 | 极高 | 可成长为顶级战鱼 |
| 暗红 | 变异级 | 完成伟业后的异化潜力 |
| 黑 | 传说级 | 极罕见，不纳入常规公开体系 |

#### 规则说明

- 鱼灵颜色不是赛事官方评级
- 赛事方无法直接检测鱼灵颜色
- 颜色越高，只代表成长上限越高，不代表当前一定更强
- 同一颜色的鱼，也可能因训练程度、体格、经验不同而表现悬殊
- 暗红与黑属于非常规层级，不作为公众赛事常见公开分类

> **结论**：
> 鱼灵颜色是"天赋/灵魂层"的判定，不等于官方战力评分。

---

### 二、赛事评分（官方体系）

赛事评分代表一条鱼**当前可量化的公开战斗能力**。
这是赛事方用于：

- 报名审核
- 战力分级
- 对阵参考
- 赔率评估
- 档案管理

的核心公开指标。

赛事评分由赛事方根据以下可见信息综合判定：

- 物种基础
- 个体差异
- 公开战绩
- 已认证技能（仅高级赛事计入）

评分满分为 **100分**。

> **注意**：
> 赛事评分只反映"赛事方看得到的实力"，不保证等于真实战力。
> 误判、隐藏技能、特殊经历，都可能导致"低评分高战力"现象。

---

### 三、认证等级（赛事流通体系）

认证等级代表这条鱼已经通过了哪一级赛事体系认证。
认证等级决定这条鱼之后能参加哪一级赛事，也决定它是否会被系统"锁级"。

#### 认证等级定义

| 认证等级 | 含义 | 可流向赛事 |
| --------- | ---- | ---------- |
| 未认证 | 从未正式参赛 | 蓝鲸杯起步 |
| 蓝鲸杯认证 | 已完成蓝鲸杯体系认证 | 龙门杯市级赛 |
| 市级认证 | 已完成龙门杯市级赛认证 | 赛区决赛 |
| 赛区认证 | 已完成赛区级认证 | 全国冠军赛 |
| 全国认证 | 已完成全国赛认证 | 斗王挑战 |

#### 核心规则

- 认证等级属于**赛事履历**，不是天赋判定
- 一条鱼一旦获得更高一级认证，即**不可再降级参赛**
- 锁定的是鱼，不是团队成员
- 团队成员可以更换其他未认证鱼参加低级赛事，但同一条鱼不能回头"炸鱼"

> **结论**：
> 认证等级是"赛事流通身份证"，不是鱼本身的灵魂等级，也不是当前战力评分。

---

## 第三部分：竞技单位定义（统一术语）

为避免后续规则、正文、公告页混用，术语统一如下：

---

### 一、成员

成员，指具体参赛的人类个体。

- 每个成员必须完成实名登记
- 每名成员可归属于一个团队
- 蓝鲸杯允许单人团队或双人团队
- 更高等级赛事是否允许双人组队，由该级赛事单独规定

---

### 二、团队

团队，是赛事中的**唯一排名单位**。

无论团队由 1 人还是 2 人组成，进入赛场后都以"1个团队"的身份结算：

- 胜负
- 名次
- 晋级资格
- 团队积分
- 奖励归属

> **重点**：
> 蓝鲸杯的混战不是"4个人乱斗"，而是"4个团队混战"。

---

### 三、战鱼

战鱼，指团队名下登记并允许上场参赛的鱼。

战鱼分为两层：

- **登记鱼池**：团队报名时登记的合法参赛鱼
- **本场出战鱼**：本场实际派出的鱼

不同赛事对"登记鱼池数量"和"单场上场数量"要求不同。

---

### 四、参赛单位

参赛单位 = 团队
不是成员个人，也不是单条鱼。

以后涉及以下项目时，一律按团队结算：

- 抽签
- 对阵
- 排名
- 晋级
- 团队积分
- 团队战绩

---

## 第四部分：场地体系

### 一、场地规格（正式比赛通用）

所有正式斗鱼竞技赛事，均使用标准化战斗场地。

| 项目 | 规格 |
| ---- | ------------------------- |
| 面积 | 标准足球场大小 |
| 高度 | 约五层楼高（≈15米） |
| 形状 | 立方体密封空间 |
| 容量 | 约100+条鱼同场极限负载 |
| 环境模拟 | 水循环、温控、光照、造流、遮蔽物、材质构造共同完成 |

---

### 二、48主题场地分类

| 类别 | 数量 | 代表主题 | 战术倾向 |
| ----- | ---- | -------- | -------- |
| 自然水域 | 8 | 亚马逊丛林、急流险滩 | 伏击、消耗、自然适应 |
| 人文景观 | 8 | 沉船遗迹、城市排水系统 | 藏身、奇袭、地形利用 |
| 极端环境 | 8 | 火山表面、冰封湖底 | 温差战、环境伤害 |
| 特殊设定 | 8 | 暗河洞穴、水母群落 | 干扰、感知战 |
| 竞技专用 | 8 | 标准训练池、环形赛道 | 纯实力、速度博弈 |
| 稀有/传说 | 8 | 龙宫遗迹、混沌之眼 | 极限规则、异常变量 |

---

### 三、环境规则（核心机制）

每个主题场地拥有 **3—5条强制环境规则**。
这些规则由场地本身决定，所有团队必须遵守。

#### 示例：T17 火山表面

| 规则 | 效果 |
| ---- | ----- |
| 温跃层 | 水体分冷水区与暖水区 |
| 热障 | 高温区持续造成环境伤害 |
| 暗流 | 暖水流方向周期变化 |
| 熔岩信号 | 场地亮纹提示环境变化 |

#### 示例：T34 环形赛道

| 规则 | 效果 |
| ---- | ----- |
| 赛道边界 | 鱼沿固定路径更容易加速 |
| 追逐加成 | 追击前方目标时获得速度优势 |
| 内道优先 | 内道抢位拥有路线优势 |
| 变道惩罚 | 离开赛道进入中央区会失去加成 |

> **规则结论**：
> 不懂场地规则的团队，会把高评分战鱼打成废物。
> 懂规则的团队，可能让普通鱼打出超预期效果。

## 第五部分：鱼类能力系统

### 一、能力来源

鱼的竞技能力由以下几部分共同构成：

1. **物种基础**
2. **个体体格**
3. **鱼灵潜力**
4. **训练所得技能**
5. **实战经验**
6. **场地适应力**
7. **团队配合度**

---

### 二、技能类型

| 类型 | 功能 | 示例 |
| ---- | ---- | ----- |
| 攻击技 | 直接造成伤害 | 冲撞、撕咬、穿刺 |
| 防御技 | 减伤或护体 | 鳞甲硬化、护盾 |
| 控制技 | 限制敌方行动 | 致盲、缠绕、眩晕 |
| 机动技 | 调整位置或位移 | 加速、跳跃、潜游 |
| 辅助技 | 增益、治疗、驱散 | 快速止血、加速同伴 |
| 被动技 | 永久生效，不需主动施放 | 高压耐受、血腥感知 |

---

### 三、技能稀有度

| 稀有度 | 含义 | 常见来源 |
| ------- | ---- | -------- |
| 普通 | 基础常见能力 | 天生、常规训练 |
| 稀有 | 需要特殊条件激活 | 血统、深度训练 |
| 传说 | 极罕见能力 | 极端经历、伟业觉醒 |

---

### 四、技能是否计入赛事评分

技能不总是公开信息。
低级赛事允许"底牌存在"，高级赛事才逐步透明化。

| 赛事级别 | 技能是否计入评分 | 说明 |
| --------- | ---------------- | ---- |
| 蓝鲸杯 | ❌ 不计入 | 视为团队隐藏底牌 |
| 龙门杯市级赛 | ✅ 部分计入 | 需通过基础技能认证 |
| 赛区决赛 | ✅ 计入 | 主要技能公开 |
| 全国冠军赛 | ✅ 计入 | 技能体系基本透明 |
| 斗王挑战 | ✅ 全面计入 | 顶级公开战力对决 |

> **结论**：
> 蓝鲸杯中，低评分鱼可能藏有高威胁技能。
> 这不是漏洞，而是入门赛事允许存在的信息差。

## 第六部分：赛事评分体系（官方体系）

---

### 一、评分区间（7级）

| 等级 | 分数区间 | 定位 |
| ---- | -------- | ---- |
| D | 1—20 | 入门级 |
| C | 21—40 | 普通竞技级 |
| B | 41—55 | 改良竞技级 |
| A | 56—70 | 赛级 |
| S | 71—85 | 顶级赛级 |
| SS | 86—95 | 传说赛级 |
| SSS | 96—100 | 神话级 |

> **说明**：
> SSS级在公开赛事中极其罕见，通常只存在于理论、历史记录或极少数异常个体中。

---

### 二、蓝鲸杯预审评分公式

蓝鲸杯使用"预审评分"，只看公开信息，不计隐藏技能：

```text
蓝鲸杯预审评分 = 物种基础分 + 个体差异分 + 公开战绩分
```

---

### 三、物种基础分

| 物种层级 | 代表鱼种 | 基础分 |
| --------- | -------- | ------ |
| 顶级赛级 | 顶级龙鱼、成鱼规格巨暹罗鲤 | 50 |
| 赛级 | 普通龙鱼、血鹦鹉、银龙、青龙 | 40 |
| 改良竞技级 | 改良罗非、地图鱼、汗鲨 | 28 |
| 普通竞技级 | 草鱼、青鱼、大板鲫、普通鲤鱼 | 20 |
| 入门级 | 普通鲫鱼、鳊鱼 | 12 |
| 观赏级 | 金鱼、孔雀鱼等小型观赏鱼 | 5 |

#### 亚成鱼规格修正

若某些高规格鱼处于亚成阶段，赛事方按"亚成规格"降低基础分。

例如：

- 巨暹罗鲤亚成鱼：基础分按规格折损后重新评估
- 但是否被赛事方准确认出，取决于品种识别是否成功

---

### 四、个体差异分（建议范围：-15 到 +20）

| 修正项 | 情况 | 修正分 |
| ------- | ---- | ------ |
| 血统纯度 | 野生原种 | +10 |
| 血统纯度 | 多代纯化 | +5 |
| 血统纯度 | 普通繁育/无证书 | -5 |
| 体格 | 超出同种均值30%以上 | +10 |
| 体格 | 超出均值15%以上 | +5 |
| 体格 | 低于均值明显 | -3 至 -5 |
| 健康 | 无伤病 | +5 |
| 健康 | 轻微旧伤 | +0 |
| 健康 | 当前有明显伤病 | -5 至 -10 |

---

### 五、公开战绩分

| 维度 | 规则 | 上限建议 |
| ----- | ---- | -------- |
| 参赛场次 | 每场+1 | +10 |
| 胜率修正 | 胜率换算 | +15 |
| 晋级记录 | 每次晋级+2 | +10 |
| MVP记录 | 每次+2 | +6 |

> **说明**：
> 蓝鲸杯阶段，战绩分通常不会太高。
> 高级赛事中，战绩分对总评分影响会逐步放大。

---

### 六、高级赛事综合评分

龙门杯及以上赛事，评分公式升级为：

```text
高级赛事综合评分 = 物种基础分 + 个体差异分 + 公开战绩分 + 已认证技能分
```

---

### 七、品种误判规则

赛事方允许存在"品种误判"。
如果某条鱼被错误认定品种，则评分按**赛事方认定结果**计算，而不是按真实物种计算。

#### 例：红焰

| 项目 | 数据 | 得分 |
| ----- | ---- | ---- |
| 赛事方认定品种 | 普通鲤鱼 | 20 |
| 血统/来源认定 | 市场通货 | -5 |
| 体格优势 | 超出同类显著 | +8 |
| 健康修正 | 轻微旧伤 | +0 |
| 战绩 | 白纸 | +0 |
| **蓝鲸杯预审评分** | — | **23分（C级）** |

#### 三重认知差

- **赛事方认知**：普通鲤鱼，23分，C级
- **陆沉当前认知**：红色鱼灵的鲤鱼，知道强，但不知道具体强在哪
- **真实本质**：巨暹罗鲤亚成鱼，红品鱼灵，具备极强潜力与隐藏技能

> **结论**：
> 品种误判合法存在，不构成违规。
> 这也是蓝鲸杯阶段最重要的信息差来源之一。

## 第七部分：赛事准入与流通规则

---

### 一、赛事准入上限

不同赛事对战鱼等级存在公开准入上限。

| 赛事 | 准入上限 |
| ---- | -------- |
| 蓝鲸杯 | A级及以下 |
| 龙门杯市级赛 | S级及以下 |
| 赛区决赛 | SS级及以下 |
| 全国冠军赛 | 不设上限 |
| 斗王挑战 | 不设上限 |

> **说明**：
> 这里的"级"指赛事评分等级，不是鱼灵颜色。

---

### 二、认证锁定规则

一条鱼一旦完成更高一级赛事认证，即不可降级参赛。

#### 流动路径

```text
未认证
→ 蓝鲸杯认证
→ 市级认证
→ 赛区认证
→ 全国认证
→ 斗王挑战资格鱼
```

#### 锁定规则

- 获得蓝鲸杯认证 → 不可再参加蓝鲸杯
- 获得市级认证 → 不可再参加蓝鲸杯与龙门杯市级赛
- 获得赛区认证 → 不可再回流低级赛事

---

### 三、报名鱼池与出场鱼池

#### 蓝鲸杯

- 每支团队最多登记 4 条战鱼
- 每场最多派出 4 条战鱼
- 也可只派 1—3 条

#### 更高级赛事

由对应赛事规则决定登记上限与出场上限。

## 第八部分：蓝鲸杯赛制（社会公开赛）

---

### 一、赛事定位

蓝鲸杯是战鱼竞技体系中的**社会公开入门赛**。
面向普通爱好者开放，是绝大多数民间团队进入正式竞技体系的第一道门槛。

---

### 二、参赛单位

- 参赛单位：团队
- 团队人数：1—2名成员
- 每支团队最多登记4条战鱼
- 单场最多派出4条战鱼

> **重点**：
> 双人组队不是"双人一起各算各的"，而是"两人共同组成1支团队"。

---

### 三、入围赛整体结构

| 项目 | 规则 |
| ---- | ---- |
| 入围赛时间 | 3月16日—3月31日 |
| 赛区数量 | 16个赛区 |
| 每赛区团队数 | 64支团队 |
| 每赛区成员容量 | 最多128名成员 |
| 每赛区晋级名额 | 6支团队 |
| 正赛总晋级 | 96支团队 |
| 正赛时间 | 4月11日开始 |
| 冠军奖金 | 50万 |

> **备注**：
> 正赛最终人数、团队换算、内部名额折算方式，以官方公告页为准。
> 总纲负责定义结构逻辑，具体名额口径可在公告中按当届执行版本公示。

---

### 四、单场对阵形式

蓝鲸杯单场采用：

```text
1团队 VS 1团队 VS 1团队 VS 1团队
```

即四支团队同时入场混战。

#### 单场基本参数

| 项目 | 规则 |
| ---- | ---- |
| 入场团队数 | 4支 |
| 每队最多出鱼 | 4条 |
| 最大同场鱼数 | 16条 |
| 最小同场鱼数 | 4条 |
| 单场时长 | 30分钟 |
| 场地 | 抽签随机主题场地 |

---

### 五、超级计算机保护机制

蓝鲸杯不以"鱼死亡"为代价决胜。
赛事方使用超级计算机进行实时生命监控。

#### 触发规则

当某条鱼即将承受致命伤害时：

1. 系统判定其生命状态越界
2. 自动触发保护程序
3. 将该鱼强制移出赛场
4. 该鱼判定为"本场出局"

#### 结果说明

- 鱼出局 ≠ 团队立刻淘汰
- 只有团队所有出战鱼全部出局，团队才算淘汰

---

### 六、单场名次判定

若 30 分钟内只剩 1 支团队仍保有战鱼在场，则该团队为本场第1名。

若时间结束时仍有多支团队未被完全淘汰，则按以下顺序判定名次：

1. 剩余战鱼数量
2. 剩余战鱼赛事评分总和
3. 本场触发对手保护次数
4. 赛前抽签顺位

---

### 七、入围赛三轮突围制

#### 第一轮：64进16

- 64支团队分成16组
- 每组4队打一场
- 每组第1名直接晋级16强

#### 第二轮：16进8

- 16强分成4组
- 每组4队打一场
- 每组前2名晋级，共8队

#### 第三轮：8进6

- 8队分成2组
- 每组4队打一场
- 每组前3名晋级，共6队晋级正赛

> **说明**：
> 蓝鲸杯晋级以**名次**为唯一硬标准，不以积分门槛决定跨级资格。

---

### 八、蓝鲸杯积分用途（版本A：晋级靠名次，积分只作辅助）

蓝鲸杯积分存在，但**不直接决定跨级赛事资格**。

#### 积分用途

1. 同轮横向排名比较
2. 团队表现奖励
3. 赛事附加服务兑换
4. 公告页说明的资料/鱼市/录像兑换（限当届）

#### 不具备的用途

- 不直接用于"硬门槛晋级龙门杯"
- 不作为跨级晋级唯一标准

> **结论**：
> 蓝鲸杯晋级靠名次，积分只是辅助工具。

## 第十部分：反刷分与公平机制

---

### 一、核心原则

蓝鲸杯是入门赛事，必须防止职业级团队反复下沉"炸鱼"。

---

### 二，反刷分机制

#### 1. 认证锁定

更高级认证鱼不能回低级赛

#### 2. 赛事准入上限

高等级公开战鱼不得降维参赛

#### 3. 公告实名举报

团队可在规定时间内实名举报疑似违规战鱼

#### 4. 赛前战力校验

赛事方会校验报名鱼与实到鱼是否一致

#### 5. 同届限制

同一支团队/同一条鱼在同届赛事周期内不得无限重复刷场

---

### 三、同盟限制

蓝鲸杯允许临时战术同盟，但有边界：

| 规则 | 说明 |
| ---- | ---- |
| 同盟不公开 | 其他团队无法提前知道 |
| 同盟击杀无额外收益 | 防止互刷 |
| 同盟可随时破裂 | 属于战术关系，不受官方保护 |

---

## 第十一部分：场外势力干预风险

> 本部分不是赛事官方规则，而是对现实环境风险的说明。

---

### 一、官方机制

- 官方只负责赛事审查、实名举报、战鱼校验
- 官方不负责处理场外组织恩怨

---

### 二、民间风险

- 民间鱼圈存在黑市、假证、替鱼，放水、情报买卖
- 高热团队可能遭遇舆论攻击或匿名举报

---

### 三、自然会风险（反派组织）

自然会**不是赛事官方**，也没有公开执法权。
但它会以场外组织的身份，对赛事中的异常人物和异常战鱼进行接触、跟踪、试探、渗透或打击。

#### 自然会可能采取的手段

- 匿名举报
- 操纵舆论
- 伪装成买家/记者/赞助人接触团队
- 提供情报诱导
- 私下招募或威胁能力者

> **结论**：
> 自然会属于场外反派势力，它能影响赛事生态，但不属于赛事裁判体系。

---

## 第十二部分：核心结论（总纲摘要）

斗鱼竞技的底层逻辑可以概括为：

1. **鱼灵颜色** 决定潜力上限
2. **赛事评分** 决定公开战力分级
3. **认证等级** 决定赛事流通路径
4. **团队** 是唯一排名单位
5. **蓝鲸杯晋级靠名次，不靠积分门槛**
6. **低评分不等于低战力，信息差始终存在**
7. **自然会是场外反派组织，不属于官方规则系统**
