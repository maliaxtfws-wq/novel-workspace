## 第一部分:竞技定位

斗鱼竞技是一种以**观赏鱼**(以龙鱼、血鹦鹉、暹罗鲤等具有战斗智能的鱼种为主)进行**对抗性战斗**的水族竞赛运动。

核心卖点:

```
场地主题 × 鱼类技能 × 选手指挥 = 战略深度
```

三个维度的自由组合,使得"以弱胜强"成为可能--一个普通打工人,也可以凭借对地形和鱼性的理解,战胜拥有顶级鱼种但不懂战术的对手。

---

## 第二部分:竞技单位

### 选手

- **参赛主体**:个人或双人组队(不同赛事有不同要求)
- **选手指挥方式**:通过「声波指令」远程控制鱼的攻守节奏(类似宝可梦的指令系统)
- **选手能力**:无强制要求,但顶级选手往往具备以下至少一项:
  - 超强记忆力(记住对手过往所有战术)
  - 地形专家(对特定场地有深刻理解)
  - 鱼性理解(能精准判断鱼的体力和情绪)
  - 超能力(感知鱼的内在状态)

### 鱼

- **参赛资格**:经过「鱼类战力认证」的鱼方可参赛
- **认证标准**:品级(颜色)+ 技能数量 + 伤病记录
- **品级(颜色体系)**:

| 颜色 | 品级 | 技能上限 | 灵智 | 参战资格 |
|------|------|---------|------|---------|
| 白 | 0 | 0 | 本能 | 禁止参赛(无战力) |
| 绿 | 0 | 0 | 本能 | 禁止参赛(无技能) |
| 蓝 | 1 | 1 | 本能 | 蓝鲸杯资格赛以上 |
| 紫 | 2 | 2 | **觉醒灵智** | 正式比赛 |
| 金 | 3 | 3 | 灵智 | 正式比赛 |
| 红 | 4 | 4+1被动 | 灵智 | 市级赛以上 |
| 暗红 | 4+1 | 4+1被动 | 高等灵智 | 全国赛以上 |

## 第三部分:场地体系

### 场地规格(所有正式比赛统一)

| 项目 | 规格 |
|------|------|
| **面积** | 标准足球场大小 |
| **高度** | 约五层楼高(≈15米) |
| **形状** | 立方体密封空间 |
| **单场容量** | 约100+条鱼同场 |
| **环境模拟** | 通过水循环、温控、光照、造流等系统实现主题环境 |

### 48场地主题分类

| 类别 | 数量 | 代表主题 | 战术倾向 |
|------|------|---------|---------|
| 自然水域 | 8 | 亚马逊丛林T01、急流险滩T06 | 伏击、耐力、自然适应 |
| 人文景观 | 8 | 沉船遗迹T09、城市排水T10 | 藏身、奇袭、地形利用 |
| 极端环境 | 8 | 火山表面T17、冰封湖底T18 | 温差战、环境伤害 |
| 特殊设定 | 8 | 水母群落T28、暗河洞穴T23 | 干扰、感知战 |
| 竞技专用 | 8 | 标准训练池T33、环形赛道T34 | 纯粹实力、速度 |
| 稀有/传说 | 8 | 龙宫遗迹T41、混沌之眼T48 | 规则变异、极限挑战 |

### 地形环境规则(核心机制)

每个主题场地都有**3-5条强制环境规则**,所有选手必须遵守:

**示例--火山表面(T17):**

| 规则 | 效果 |
|------|------|
| 温跃层 | 水体分两层:上层冷水区(0-5米)、下层暖水区(5米以下) |
| 热障 | 热血型鱼(红品以上)进入下层暖水区时,速度+30%,但每30秒受到1点热伤害 |
| 冷缓 | 冷血型鱼进入上层冷水区时,所有属性-20% |
| 暗流 | 底部暖水流会形成不规则暗流,推力方向每30秒变换一次 |
| 熔岩信号 | 场地四周的"熔岩纹"会在战斗开始后每10分钟亮起一次,提示底层水流变换方向 |

> **战术意义**:不了解规则的选手会盲目让自己的主力鱼冲进下层,然后被热障反复烧血;懂规则的选手会让热血鱼在下层速战速决,冷血鱼则全程留在上层。

**示例--环形赛道(T34):**

| 规则 | 效果 |
|------|------|
| 赛道边界 | 底部有环形凹槽,鱼会自动沿赛道巡游 |
| 追逐加成 | 在赛道上追击前方鱼类时,速度+25% |
| 内道优先 | 占据内道的鱼,在交汇点有优先通过权 |
| 变道惩罚 | 离开赛道进入中央区域时,所有属性-15%(开阔区无遮蔽) |
| 甩尾冲撞 | 在弯道处利用甩尾产生的离心力发动攻击,伤害+20% |

## 第四部分:鱼类技能系统

### 技能上限规则

- 每条鱼的**技能数量**取决于品级(颜色)
- **蓝品**最多1个技能
- **紫品**最多2个技能
- **金品**最多3个技能
- **红品**最多4个技能 + 1个进化后获得的**被动技能**

### 技能分类

| 类型 | 功能 | 示例 |
|------|------|------|
| **攻击技** | 直接造成伤害 | 冲撞、撕咬、穿刺 |
| **防御技** | 减少自身受到的伤害 | 鳞甲硬化、护盾、减伤护膜 |
| **控制技** | 限制对手行动 | 冰冻、缠绕、致盲、眩晕 |
| **机动技** | 改变位置或状态 | 加速、闪现、跳跃、潜水 |
| **辅助技** | 治疗或增益队友 | 治疗、加速队友、驱散debuff |
| **被动技** | 永久生效,无需触发 | 血腥味感知、高压耐受 |

### 技能稀有度

每个技能有稀有度评级(与鱼的品级无关):

| 稀有度 | 含义 | 获得方式 |
|--------|------|---------|
| **普通** | 常见技能,大多数鱼可学习 | 天生/训练 |
| **稀有** | 需要特殊血统或训练激活 | 血统遗传/深度训练 |
| **传说** | 几乎不可见,与鱼的"经历"相关 | 极端经历觉醒/传承 |

> 被动技能的来源只有一个:通过「不可思议的伟业」进化后自动获得,不可选择,不可更换。

### 技能组合机制(高级规则)

| 机制 | 说明 |
|------|------|
| **技能连携** | A鱼使用控制技命中后,B鱼的下一击必定命中 |
| **阵型buff** | 特定鱼种组合在场时,激活额外属性加成 |
| **诱饵战术** | 用低价值鱼引诱对手暴露关键技能的冷却时间 |
| **技能压制** | 某些技能可以"克制"其他技能(如:驱散可以解除控制) |

### 第五维:蓝鲸杯(社会公开赛)

**定位**:入门级比赛,面向所有普通爱好者

| 项目 | 规则 |
|------|------|
| **入围赛时间** | 3月16日 - 3月31日(共16天,每天1个赛区) |
| **入围赛规模** | 16个赛区,每赛区128人 |
| **正赛时间** | 4月11日开始(休整10天后) |
| **正赛规模** | 96名社会晋级选手 + 32名蓝星内部名额,共128人 |
| **正赛赛制** | 淘汰赛 |
| **冠军奖金** | 50万 |

---

#### 入围赛(3月16日-3月31日)

**赛制核心:超级计算机实时保护 + 1v1v1v1混战**

| 项目            | 规则                                          |
| ------------- | ------------------------------------------- |
| **每日参赛人数**    | 128人(1个赛区)                                  |
| **每场比赛入场人数**  | 4人同时入场（1v1v1v1）                               |
| **每场比赛入场鱼数**  | 每位选手最多可派出**4条鱼**同时参战,也可以只派出**1条鱼**               |
| **同时在场鱼数**    | 最多16条鱼同场（4人×4条）,最少4条鱼同场（4人×1条）                    |
| **比赛时长**      | 30分钟/场                                      |
| **晋级名额**      | 每赛区前6名,共96人晋级正赛                             |
| **超级计算机保护机制** | 实时监控鱼的"生命状态",当鱼即将承受致命伤害时触发保护,触发保护的鱼被强制移出并判负 |
| **胜负判定**      | 保护触发=判负;最后存活的鱼获胜;若均未触发保护,按评分排名            |

> **关键战术考量**:选手派出的鱼越多,战力越强,但风险也越分散;只派1条鱼可能成为"众矢之的",但也更容易集中资源保护。

**晋级规则:**

| 阶段 | 规模 | 规则 |
|------|------|------|
| 小组赛 | 128人分成32组(每组4人) | 每组打循环积分赛,每场第1名直接晋级 |
| 复活赛 | 每场第2-3名进入复活池 | 复活池32人打成积分赛,取前N名凑满6人 |
| 晋级名额 | 每赛区前6名 | 32直接晋级+复活晋级=96人 |

> 每赛区的比赛在当天完成,第二天换下一个赛区。

**陆沉参赛时间**:3月16日(第1个赛区,浅圳本地赛区)

---

#### 蓝鲸杯正赛(4月11日开始)

| 项目 | 规则 |
|------|------|
| **参赛人数** | 128人(96名社会晋级+32名蓝星内部名额) |
| **赛制** | 淘汰赛 |
| **对阵** | 128进64→64进32→32进16→16进8→8进4→半决赛→决赛 |
| **正赛时长** | 约5-7天 |
| **冠军奖励** | 50万奖金 + 直通龙门杯市级赛 |

> **内部名额说明**:蓝星集团内部名额不占社会晋级名额,张海明、赵洋等人通过内部选拔获得参赛资格,与社会晋级选手在同一赛事中竞技。

---

> 核心设计原则:每一级比赛,选手需要掌握的维度都在增加。
> 不是"对手更强",而是"需要考虑的东西更多"。

### 第五维:蓝鲸杯(社会公开赛)

**定位**:入门级比赛,面向所有普通爱好者

| 项目 | 规则 |
|------|------|
| **入场门槛** | 无(缴费即可参赛) |
| **场地** | T33「标准训练池」(净水泥池,无任何遮蔽) |
| **参赛鱼数** | 每人最多6条 |
| **同时在场** | 最多6条(全部上场) |
| **战斗时长** | 30分钟 |
| **胜负规则** | Knockout制--时间结束时,存活鱼多的选手胜;若双方同时全部阵亡,则比较剩余鱼的总品级分数 |

**核心维度**:**硬实力**

- 纯看鱼的品级和数量,无任何策略博弈空间
- 地形无变量,不存在利用地形一说
- 不需要预备池、不需要调度

**陆沉过关方式**:

- 超能力感知对方的颜色品级
- 优先击溃对方品级最低的鱼,确保数量优势
- 红焰作为红品主力,在蓝鲸杯级别几乎是碾压级的存在

### 第四维:浅圳市龙门杯(市级赛)

**定位**:进入正式竞技体系的门槛,128人参赛

| 项目 | 规则 |
|------|------|
| **入场门槛** | 蓝鲸杯前32名 / 个人自行报名(需战力认证) |
| **场地** | 引入主题场地(从T01-T16中抽签决定) |
| **参赛鱼数** | 每人最多8条 |
| **同时在场** | 最多6条 |
| **战斗时长** | 45分钟 |
| **胜负规则** | 夺分制--场地中央有3个得分点,占据并存活超过10分钟得1分;结束时分数高者胜 |

**新增维度**:

| 维度 | 说明 |
|------|------|
| **得分点争夺** | 不需要全歼对手,占点+守点才是核心 |
| **地形利用** | 掩体可以埋伏、暗角可以偷袭、水流可以借力 |
| **诱饵战术** | 用低品鱼去占点,主力埋伏在侧翼 |
| **伤病机制** | 鱼一旦受伤,下场后需休息1场才能再次使用 |

**陆沉过关方式**:

- 超能力感知掩体里是否有埋伏(别人看不到但他能)
- 超能力判断对方主力鱼的颜色明暗(体力高低)
- 在对方换鱼的间隙发动进攻

### 第三维:赛区龙门杯(地区级赛)

**定位**:32人参赛,争夺全国赛门票

| 项目 | 规则 |
|------|------|
| **入场门槛** | 龙门杯市级前16名 |
| **场地** | 引入全主题场地(T01-T40,含竞技专用地形) |
| **参赛鱼数** | 每人最多10条 |
| **同时在场** | 最多6条(剩下4条在预备池) |
| **战斗时长** | 60分钟,分3轮,每轮20分钟 |
| **胜负规则** | 轮转积分制--每轮独立计分,三轮总分最高者胜

**新增维度**:

| 维度 | 说明 |
|------|------|
| **预备池** | 同时在场最多6条,剩下4条在预备池待命 |
| **调度窗口** | 每场只有3次换鱼机会(把预备池的鱼派上场),用还是不用?什么时候用? |
| **疲劳机制** | 鱼战斗超过10分钟后,体力开始下降,每5分钟属性-10% |
| **伤病标记** | 受伤的鱼下场后必须休息至少1轮 |
| **品级压制** | 蓝品以下(含蓝品)的鱼,无法对紫品以上造成有效伤害 |

**赛制细节**:

```
第一轮(20分钟)
  → 选手派出第一批6条鱼上场
  → 战斗开始,20分钟后进入"中场调度窗口"(2分钟)

中场调度窗口
  → 选手可换鱼(最多换3条)
  → 受伤的鱼被收入预备池
  → 新上场的鱼视为"体力充沛"

第二轮(20分钟)
  → 继续战斗,20分钟后再次进入调度窗口

第三轮(20分钟)
  → 最后20分钟,预备池所有鱼必须全部派上场
  → 最终存活数决定胜负
```

**陆沉过关方式**:

- 超能力感知对方主力鱼的颜色明暗变化(体力消耗程度)
- 预判对方在第几轮派出主力(从而在对方疲劳期发起进攻)
- 红焰的「领袖气质」--周围友方鱼属性+10%

### 第二维:全国冠军赛(全国32人)

**定位**:全国最顶尖的32人同台竞技,争夺"斗王"称号

| 项目 | 规则 |
|------|------|
| **入场门槛** | 各赛区前3名(共12人)+ 外卡4人 + 卫冕冠军 |
| **参赛鱼数** | 每人最多12条 |
| **同时在场** | 最多6条 |
| **赛制** | 多日赛制,共5场(3天),累计积分定胜负 |
| **单场规则** | 每场比赛规则随机(见下),开赛当日早晨公布 |

**新增维度(规则盲盒)**:

| 盲盒规则 | 说明 |
|---------|------|
| **完全信息战** | 双方选手的鱼名单、顺序完全公开 |
| **暗牌战** | 双方只公开鱼的品级,不公开具体技能 |
| **迷雾战** | 双方不公开任何信息 |
| **单挑王** | 每场只能派1条鱼上场,一局定胜负 |
| **群雄逐鹿** | 32人同时入场,混战,最后存活的选手晋级 |
| **田忌赛马** | 三战两胜,每场由品级低的鱼先上,品级高的后上 |

**新增机制(鱼市交易)**:

| 机制 | 说明 |
|------|------|
| **积分租借** | 用累积积分可以在"鱼市"临时租借其他选手的鱼(租借费:500积分/条) |
| **情报交易** | 可以购买其他选手过往比赛的录像和战术分析 |
| **鱼的状态披露** | 租借的鱼不披露其"脾性",用不好可能反噬 |

**陆沉过关方式**:

- 红焰在陌生场地中能感知"天灾"前夕的异常反应
- 陆沉根据鱼的体力明暗判断"调度窗口"的最佳时机
- 「信息差」优势--在暗牌战中,超能力让他知道对手不知道的东西

## 第八部分:赛事准入与反刷分机制

### 核心原则

蓝鲸杯是面向普通爱好者的入门级赛事,必须防止高水平选手("炸鱼")垄断晋级名额,维护赛事公平。

---

### 一、认证锁定制度

每条鱼在首次参加更高级别赛事后,将被「认证锁定」,不可降级参赛。

| 认证状态 | 可参赛 | 不可参赛 |
|---------|-------|---------|
| 未认证(白纸) | 蓝鲸杯 | 任何更高级别赛事 |
| 蓝鲸杯认证 | 龙门杯市级赛 | 再次参加蓝鲸杯 |
| 市级认证 | 赛区决赛 | 蓝鲸杯/龙门杯 |
| 赛区认证 | 全国冠军赛 | 蓝鲸杯/龙门杯/赛区 |
| 全国认证 | 斗王挑战 | 所有限制级别 |

> **说明**:认证锁定的是鱼,不是选手。选手可以用其他未认证的鱼参加低级赛事,但同一条鱼不能降级。

---

### 二、蓝鲸杯参赛限制

| 限制项 | 规则 |
|-------|------|
| 报名鱼数 | 每位选手最多报名3条鱼参加蓝鲸杯(其余为备用,不计入认证) |
| 认证锁定 | 报名的3条鱼在蓝鲸杯赛季期间(至龙门杯开赛前)被认证锁定 |
| 存活要求 | 只有在入围赛中存活的鱼(未被击败)才能获得「晋级资格」 |
| 被淘汰鱼 | 入围赛被淘汰的鱼,可以继续参加当届蓝鲸杯其他场次,但积分不能用于晋级龙门杯 |

> **关键逻辑**:不是"参赛"就锁定,而是"报名且存活"才锁定。如果你的鱼第一场就被淘汰了,它还是白纸,可以参加下届蓝鲸杯。

---

### 三、积分分级制度

不同级别赛事的积分有不同的用途,防止低级赛事积分被滥用于高级场景。

| 积分来源 | 积分用途 |
|---------|---------|
| **蓝鲸杯积分** | 1 晋级龙门杯(需500分)2 鱼市租借(每100分可租1条鱼1场)3 低额兑换(1分=0.1元) |
| **龙门杯积分** | 1 晋级赛区决赛(需800分)2 鱼市租借3 中额兑换(1分=1元) |
| **赛区积分** | 1 晋级全国冠军赛2 高额兑换(1分=10元) |

> **反刷分逻辑**:在低级赛事刷积分→兑换现金效率极低;想赚钱必须去高级赛事;想刷晋级资格,刷完还是得往上走,不能蹲低级赛。

---

### 四、双人同盟限制

入围赛允许选手临时组成"非正式同盟",但有以下限制:

| 规则 | 说明 |
|------|------|
| 同盟内击杀不计分 | 同盟选手之间互相击杀,不计入得分 |
| 同盟数量上限 | 最多4人组成临时同盟 |
| 同盟关系不公开 | 比赛中其他选手无法得知谁与谁是同盟 |

> **防止互刷**:故意让盟友全灭、独吞击杀分的做法失效;但真实的战术同盟仍然被允许。

---

### 五、自然会监视机制

自然会有动机去抓"炸鱼"行为--他们希望阻止民间高手流入"斗王计划"。

| 触发条件 | 后果 |
|---------|------|
| 选手使用明显超出蓝鲸杯水平的技术/装备 | 自然会介入调查,必要时取消资格 |
| 同一选手反复用同一批鱼在不同届蓝鲸杯刷分 | 自然会标记为"职业选手",禁止参赛 |
| 选手被举报"疑似炸鱼" | 自然会调取该选手历史比赛录像评估 |

> **注意**:这是自然会的主动行为,不是赛事官方的规则。自然会可能借机接近/监视陆沉(hook_014)。

---

### 六、漏洞与补丁

| 漏洞 | 补丁 |
|------|------|
| 高水平选手用备用鱼在蓝鲸杯反复刷分 | 认证锁定+赛季限制,同一选手用同条鱼只能参加一届蓝鲸杯 |
| 双人同盟互刷击杀分 | 同盟内击杀不计分,但允许真实战术同盟 |
| 用积分兑换现金后在低级赛事反复参赛 | 兑换比例极低(蓝鲸杯1分=0.1元),刷分收益不如正常工作 |
| 买通低级选手故意输分 | 自然会监视+举报机制,高频输分会被标记调查 |

## 第九部分:赛事分级准入与鱼品级体系

---

### 一、鱼品级名称(7级)

取消原有的ABCD体系,改为更精细的7级命名:

| 品级名称 | 评分区间 | 定位 | 参赛资格 |
|---------|---------|------|---------|
| **D** | 1-30分 | 入门级观赏鱼 | 蓝鲸杯 |
| **C** | 31-50分 | 普通竞技鱼 | 蓝鲸杯 |
| **B** | 51-65分 | 改良竞技鱼 | 蓝鲸杯 |
| **A** | 66-80分 | 赛级纯种鱼 | 蓝鲸杯 |
| **S** | 81-90分 | 顶级赛级鱼 | 龙门杯市级赛及以上 |
| **SS** | 91-99分 | 传说级赛鱼 | 赛区决赛及以上 |
| **SSS** | 100分 | 神话级(传说存在) | 全国冠军赛及以上,不限定 |

> **SSS说明**:现实中几乎不存在,只有理论意义。能达到SSS级的鱼,通常与渊眼洞窟的放射性矿物影响有关,或是经历了极端环境的考验(暗红进化)。

---

### 二、鱼类赛事评分计算方法

#### 总分构成(三项合计,满分100分)

```
赛事评分 = 物种基础分 + 个体差异分 + 战绩加成
```

#### 1 物种基础分

| 物种分类 | 代表鱼种 | 基准分 |
|---------|---------|--------|
| **顶级赛级** | 顶级龙鱼(血红龙、辣椒红龙)、巨暹罗鲤(成鱼规格) | 50分 |
| **赛级** | 普通龙鱼、银龙、青龙、血鹦鹉 | 40分 |
| **改良竞技级** | 改良罗非、地图鱼、汗鲨 | 28分 |
| **普通竞技级** | 草鱼、青鱼、大板鲫 | 20分 |
| **入门级** | 普通鲫鱼、鳊鱼 | 12分 |
| **观赏级** | 金鱼、孔雀鱼等小型观赏鱼 | 5分 |

> **亚成鱼规格说明**:某些物种在亚成阶段体型远未达标,赛事方会按「亚成鱼规格」降低基准分。例:巨暹罗鲤亚成鱼(<50cm/<20kg),基准分按×0.7计算,即35分。

#### 2 个体差异分(上限+20,下限-15)

| 修正项 | 情况 | 修正分 |
|-------|------|--------|
| 血统纯度 | 野生原种 | +10 |
| 血统纯度 | 纯种繁殖(多代纯化) | +5 |
| 血统纯度 | 杂交改良(有血统证书) | +0 |
| 血统纯度 | 市场通货(无证书) | -5 |
| 体格 | 超出同种平均体型30%以上 | +10 |
| 体格 | 超出同种平均体型15%以上 | +5 |
| 体格 | 正常平均体型 | +0 |
| 体格 | 低于同种平均体型 | -3 |
| 体格 | 明显弱小于同龄同种 | -5 |
| 健康状态 | 完全健康,无伤病记录 | +5 |
| 健康状态 | 轻微伤病史(已痊愈) | +0 |
| 健康状态 | 陈旧伤病(影响部分机能) | -5 |
| 健康状态 | 当前有伤病或慢性问题 | -10 |

#### 3 技能加成(按赛事级别计入)

技能是鱼通过后天训练获得的"隐性底牌",不是天生的展示能力。技能的存在本身不会被赛事方计入评分,只有在更高级别赛事中才会被认证和记录。

| 赛事级别 | 技能是否计入评分 | 技能加分上限 |
|---------|----------------|------------|
| 蓝鲸杯 | ❌ 不计入(技能是隐藏底牌) | - |
| 龙门杯市级赛 | ✅ 计入 | +10分 |
| 赛区决赛 | ✅ 计入 | +15分 |
| 全国冠军赛 | ✅ 计入 | +20分 |

> **为什么蓝鲸杯不计入?** 蓝鲸杯是入门级赛事,面向普通爱好者。很多人的鱼是养了很久的感情鱼,训练出了独特的技能,但这些技能不会被赛事方公开--因为它们是选手的私人财产,是比赛中的"底牌"。
>
> **这意味着:** 一条赛事评分只有23分的C级鲤鱼,可能藏着四个技能;一条赛事评分61分的B+级龙鱼,可能一个技能都没有。高级别赛事中,这种信息差会逐渐缩小(因为技能会被认证记录)。

#### 4 战绩加成(上限+41分)

| 维度 | 加分规则 | 上限 |
|------|---------|------|
| 参赛场次 | 每场+1分 | +10分 |
| 胜率加成 | 胜率×15 | +15分 |
| 淘汰赛加成 | 每次晋级+2分 | +10分 |
| MVP次数 | 每次+2分 | +6分 |

#### 5 特殊规则:品种误判

赛事方对鱼的品种认定可能出现误判。这种情况下,评分按「赛事方认定的品种」计算,但鱼的「真实战力」不受影响。

```
例:红焰
  → 赛事方认定:普通鲤鱼,市场通货 → 物种基准分20分
  → 真实品种:巨暹罗鲤亚成鱼 → 真实战力远超C级
  → 赛事评分:23分(C级,蓝鲸杯版本)
  → 真实鱼灵:红色 → 拥有4个稀有技能,实际战力可达S级以上
```

> 品种误判是合法现象,不算违规。选手可利用误判参赛,但一旦鱼的真正实力暴露,赛事方不会追溯调整历史成绩。

### 二、赛事准入等级限制

每级赛事对参赛鱼的品级有硬性上限:

| 赛事 | 准入最高品级 | 说明 |
|------|-------------|------|
| **蓝鲸杯** | A级以下(A/C/B/D) | S级以上鱼禁止参加,发现则取消资格 |
| **龙门杯市级赛** | S级以下(A/C/B/D/S) | SS级以上禁止参加 |
| **赛区决赛** | SS级以下(A/C/B/D/S/SS) | SSS级禁止参加 |
| **全国冠军赛** | 不限定 | 任何品级均可参加 |

### 三、准入检查机制

| 检查点 | 执行方式 |
|-------|---------|
| 报名时 | 赛事方根据鱼的血统证书、技能认证记录进行初评 |
| 入场时 | 赛前对参赛鱼进行"战力检测"(和认证评分对比,防止带高级鱼冒名顶替) |
| 举报 | 其他选手可在比赛开始前1小时发起举报(实名) |
| 自然会 | 有权对"疑似炸鱼"行为主动介入 |

### 四、跨级参赛的限制(认证锁定补充)

结合第八部分的认证锁定,完整规则如下:

```
鱼品级流动规则:

未认证鱼 → 只能从蓝鲸杯开始
蓝鲸杯认证 → 升级为"蓝鲸杯认证鱼",可参加龙门杯市级赛
市级认证   → 升级为"市级认证鱼",可参加赛区决赛
赛区认证   → 升级为"赛区认证鱼",可参加全国冠军赛
全国认证   → 可参加斗王挑战

认证锁定:
  → 鱼一旦获得某级认证,即不可再参加该级及以下赛事
  → 例:获得蓝鲸杯认证的鱼,不得再次参加蓝鲸杯
  → 例:获得市级认证的鱼,可以继续参加蓝鲸杯吗?不可以,认证锁定不可逆
```
