# 蓝鲸杯参赛流程：报名—认证—检录—比赛

> 2026-04-05 新建 v1.0
> 目的：把"普通人怎么报上名"写实，支撑第5章以后比赛流程

---

## 一、参赛门槛

| 条件 | 要求 |
|------|------|
| 年龄 | 18岁以上 |
| 身份 | 不限地域，全国可报 |
| 参赛鱼 | 每位选手最多报3条（最多出场4条） |
| 报名费 | 200元/人 |
| 参赛鱼资格 | 需持有鱼的品种证书（或自愿承担误判风险） |

> **个人名义报名**：没有限制。任何人都可以以个人名义报名，不需要有俱乐部、赞助商、单位背书。

---

## 二、报名流程

### 线上报名（第1步：注册账号）

1. 打开蓝鲸杯官网或官方合作平台
2. 用手机号注册账号
3. 填写个人信息（姓名、身份证、联系方式）

### 线上报名（第2步：提交参赛鱼信息）

1. 上传参赛鱼的照片（正面、侧面、背部各1张）
2. 填写鱼的品种（按赛事方提供的目录选择）
3. 如有血统证书，上传证书照片（没有可以跳过）
4. 系统根据品种自动给出"参考评分"——选手可接受也可申诉

> ⚠️ **品种误判风险**：如果选手填写的品种与赛事方认定不符，以赛事方认定为准。红焰的案例：选手填"鲤鱼"，赛事方认定"普通鲤鱼"，评分23分。

### 线上报名（第3步：缴费）

- 报名费：200元/人
- 支付方式：微信/支付宝
- 缴费成功即报名成功

### 线下报名

- 前往蓝鲸杯合作报名点
- 工作人员代为填写信息
- 同样需要手机号注册账号

---

## 三、认证流程

### 入场认证（第4步：现场确认）

比赛当天，选手携带参赛鱼到达赛区：

| 步骤 | 内容 | 时长 |
|------|------|------|
| 1 | 凭身份证和报名码签到 | 5分钟 |
| 2 | 提交参赛鱼，工作人员拍照比对 | 10-15分钟/条 |
| 3 | 工作人员对鱼进行"战力初检"（外观+体型） | 10分钟/条 |
| 4 | 确认鱼的状态（有无明显伤病） | 5分钟 |
| 5 | 录入系统，生成认证编号 | 10分钟 |

> ⚠️ **认证锁定触发点**：完成入场认证后，鱼进入"认证锁定"状态，该鱼本赛季只能参加已认证的赛事级别。

### 认证费用

| 项目 | 费用 |
|------|------|
| 现场认证 | 50元/条 |
| 认证锁定工本费 | 100元/条（晋级后自动扣除） |

---

## 四、检录流程

### 每场比赛前检录

| 步骤 | 内容 | 时长 |
|------|------|------|
| 1 | 赛前30分钟，选手带鱼到场 | — |
| 2 | 裁判核验选手身份和鱼的照片 | 5分钟 |
| 3 | 确认出场鱼名单（最多4条，可少于4条） | 3分钟 |
| 4 | 鱼的状态确认（有无伤病） | 5分钟 |
| 5 | 抽签决定入场顺序 | 2分钟 |

> **出场鱼选择**：选手可以从报名的3条鱼中任选1-4条出场。策略性选择——全上还是留底牌？

## 五、入围赛流程

### 小组赛阶段

```
128人 → 分成32组（每组4人）
每组4人打循环赛，每场4人同时入场
每场第1名直接晋级
```

| 项目 | 规则 |
|------|------|
| 入场 | 4名选手+各自的出场鱼（最多4条）同时入场 |
| 同时在场 | 最多16条鱼，最少4条 |
| 时长 | 30分钟/场 |
| 晋级 | 每组每场第1名直接晋级 |
| 复活 | 每场第2-3名进入复活池 |

### 复活赛阶段

```
复活池32人 → 积分赛 → 取前N名凑满96人
```

| 项目 | 规则 |
|------|------|
| 赛制 | 积分制（胜场数+评分排名） |
| 名额 | 根据缺口数量，取前N名晋级 |
| 隐藏规则 | 赛事方综合评估可影响最终名额（见赛制公告页） |

## 六、退赛与弃权

| 情况 | 处理方式 |
|------|---------|
| 主动退赛 | 赛前1小时申请，退还报名费；赛后不退 |
| 无故弃权 | 拉入黑名单，2年内禁止参加蓝鲸杯 |
| 鱼死亡 | 可用备用鱼替换，但需要重新认证 |
| 伤病退赛 | 提供兽医证明，可保留下一届参赛资格 |

## 七、陆沉个人名义报名流程（实操示例）

```
1. 打开官网，用手机号注册
2. 上传红焰照片（正面+侧面+背部）
3. 品种选择：鲤鱼
4. 系统参考评分：23分（符合预期）
5. 缴费200元
6. 收到报名成功短信和报名码

比赛当天：
7. 携带红焰到达浅圳赛区
8. 现场认证：拍照+体型检测+状态确认
9. 缴费50元（认证费）
10. 拿到认证编号：S-0237
11. 等待入场
```
