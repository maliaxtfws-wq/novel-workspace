## 第一部分：触发条件与感知内容

### 触发条件

**鱼钩与鱼产生任何形式接触时，被动触发**
- 鱼咬钩 → 触发 ✅
- 鱼钩从侧面挂到鱼 → 触发 ✅
- 手持鱼钩触碰鱼体 → 触发 ✅

**使用代价**：鱼钩必须与鱼有物理接触

### 感知内容：鱼灵（颜色品级）

鱼灵颜色代表这条鱼的品级上限——品级越高，成长性越强，能学的技能越多。

| 鱼灵颜色 | 技能上限 | 灵智水平 | 特殊能力 |
|---------|---------|---------|---------|
| 白 | 0（不能学技能） | 本能 | 无 |
| 绿 | 0 | 本能 | 体格比普通鱼强壮15% |
| 蓝 | 1 | 本能 | 无 |
| 紫 | 2 | **觉醒灵智，有自己的思维** | 无 |
| 金 | 3 | 灵智 | **可理解动物语言** |
| 红 | 4 | 灵智 | **可学习其他生物语言，向同类发号施令** |
| 暗红 | 4+1被动 | 灵智 | **自然界不会出现，需达成不可思议的伟业后蜕变** |

## 第二部分：能力代价与限制

> 这是防止能力变万能锁的核心设定。

### 第一类：生理代价

每次触发感知后，陆沉会承受以下生理反应：

| 代价类型 | 表现 | 持续时间 |
|---------|------|---------|
| 轻微头痛 | 太阳穴两侧隐隐胀痛 | 10-30分钟 |
| 视野残影 | 视野边缘有轻微重影 | 5-15分钟 |
| 耳鸣 | 低频耳鸣，像水管里的水流声 | 5-20分钟 |
| 精神疲惫 | 触发后精力明显下降，不想说话 | 30-60分钟 |

**重度使用代价**（同时感知多条高品鱼时）：
- 剧烈头痛，伴随轻度恶心
- 视野大面积重影，几乎看不清近处物体
- 可能出现短时间单耳失聪
- 恢复时间延长至2-4小时

> **写作原则**：生理代价必须在触发后**立即体现**，不能拖延到下一场景才写。

### 第二类：信息限制

感知有明确的**信息边界**，不是万能读心术：

| 限制类型 | 具体表现 |
|---------|---------|
| **只能看到颜色/轮廓** | 看不到具体技能详情，不能判断这条鱼会什么技能 |
| **距离限制** | 水深超过3米、距离超过10米时，感知准确性大幅下降 |
| **介质限制** | 浑水、夜晚、深水，感知精度下降 |
| **同时感知多条鱼** | 两条以上高品鱼（蓝品以上）时，颜色会互相干扰，难以分辨 |
| **品种识别** | 能判断"红色灵魂"，但不一定能判断具体品种（红焰=陆沉以为的"红色鲤鱼"） |
| **技能状态** | 只能感知当前的灵智状态，不能感知"冷却时间"或"疲劳程度" |

**信息示例**：

```
触发前（现实）：一条在水下3米深处的紫品鱼在游动
触发后（感知）：视野中出现一个紫色的模糊轮廓，能看到大概体型，看不到细节
实际获得的信息：这是一条紫品鱼，大概30cm长，游动状态
实际获得不了的信息：它有什么技能？它游向哪里？它饿不饿？
```

### 第三类：成长限制

五阶段有明确的解锁边界，没解锁之前**绝对不能**做到：

| 阶段 | 名称 | 能做什么 | 不能做什么 |
|------|------|---------|---------|
| 第一阶段 | 鱼色 | 看鱼的品级上限 | 不能分辨具体技能，不能看伤病 |
| 第二阶段 | 鱼技 | 看鱼的技能痕迹（曾经学过的技能） | 不能感知正在使用的技能 |
| 第三阶段 | 鱼态 | 看鱼的体力/伤病/当前状态 | 不能看鱼的战术意图 |
| 第四阶段 | 鱼意 | 感知鱼的意图/战意/指令倾向 | 不能完全读心，只能感知情绪方向 |
| 第五阶段 | 鱼视 | 短暂共享鱼的局部视角或战场理解 | 有时间限制（每次最多30秒），一天最多用3次 |

## 第三部分：五阶段解锁条件（骨架）

### 第一阶段「鱼色」

**当前阶段**
- 触发条件：鱼钩与鱼产生任何形式接触
- 感知内容：鱼灵颜色（品级上限）
- 代价：生理代价（头痛/耳鸣/视野重影/精神疲惫）

### 第二阶段「鱼技」（解锁条件：累计触发鱼色30次以上 + 一次生死危机）

- 触发条件：仍然是被动触发，但感知深度增加
- 感知内容：鱼的"技能痕迹"——它曾经学会过什么技能（但看不到当前技能）
- 新代价：触发后会出现短暂的"技能残影"画面

### 第三阶段「鱼态」（解锁条件：第二阶段稳定后 + 长期照顾一条鱼）

- 触发条件：主动聚焦（不再是纯被动）
- 感知内容：鱼的当前状态（体力百分比、伤病情况、疲劳程度）
- 新代价：主动聚焦会消耗精力，聚焦时间越长消耗越大

### 第四阶段「鱼意」（解锁条件：第三阶段后 + 一次深度人鱼共鸣）

- 触发条件：需要与鱼建立某种"连接感"
- 感知内容：鱼的意图和情绪（战斗意愿、恐惧、愤怒、服从）
- 新代价：感知到敌意时，自己也会短暂感受到同样的情绪冲击

### 第五阶段「鱼视」（解锁条件：完成"不可思议的伟业"后）

- 触发条件：主动触发，有次数限制
- 感知内容：共享鱼的局部视角或战场理解
- 限制：每次最多30秒，每天最多3次，使用后强制休息6小时

## 第四部分：早期陆沉状态

- **感知不完整**：能感知到鱼灵颜色，但不理解颜色背后的含义
- **无法识别品种**：知道这条鱼"是红色的"，但不知道它具体是什么品种
- **不会控制**：被动触发，无法主动开启或关闭
- **隐瞒能力**：不告诉任何人

### 陆沉的认知局限（第一阶段）

```
红焰的真实身份：巨暹罗鲤亚成鱼，红品鱼灵
陆沉的认知：红色灵魂的鲤鱼（误判品种）
赛事方的认知：普通鲤鱼，市场通货（误判品种+低估品级）
```

> 陆沉的超能力告诉他"这条鱼不普通"，但他的认知还不足以判断它是巨暹罗鲤。
> 直到中期，随着超能力逐渐完整，他才会意识到红焰的真实身份。
