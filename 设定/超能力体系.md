## 一、总体升级逻辑

**五阶段能力链条：**

| 阶段 | 核心 | 一句话定义 |
|------|------|-----------|
| 第一阶段「鱼色」 | 看上限 | 看见鱼的品级与成长上限 |
| 第二阶段「鱼技」 | 看技能 | 识别鱼的技能结构与技能痕迹 |
| 第三阶段「鱼态」 | 看状态 | 读取鱼的实时状态与战斗窗口 |
| 第四阶段「鱼意」 | 看意志 | 感知鱼的情绪、意志与战斗倾向 |
| 第五阶段「鱼视」 | 看战场 | 短暂与鱼共享视角，获得战场级感知 |

**总逻辑：鱼色看"上限"，鱼技看"手段"，鱼态看"状态"，鱼意看"意志"，鱼视看"战场"。**

> 这不是五个散能力，而是一条很顺的升级链。
> 从"知道鱼有多强"→"知道鱼会什么"→"知道鱼现在怎么样"→"知道鱼想什么"→"知道鱼看见什么"。

## 二、第一阶段：鱼色（当前阶段）

### 核心定义

> 鱼钩与鱼发生物理接触时，陆沉能感知鱼灵颜色，颜色代表该鱼的品级上限、灵智层级与成长性。

### 触发条件

- 鱼钩与鱼产生任何形式接触时**被动触发**
- 包括：鱼咬钩 / 鱼钩挂到鱼 / 手持鱼钩触碰鱼体

### 感知内容

视野中浮现鱼的灵魂颜色，以及该颜色的部分轮廓感知。

### 颜色品级体系

| 颜色 | 品级 | 说明 |
|------|------|------|
| 白 / 绿 | 普通 | 罗非鱼等常见鱼 |
| 蓝 / 紫 | 中级 | 中小型战鱼 |
| 金 | 高级 | 强力战鱼 |
| **红** | 顶级 | **本章出现的命运之鱼**，像凝固的血 |
| 暗红 | 极稀有 | 需"不可思议的伟业"方可蜕变 |
| 黑 | 传说 | 几乎不存在的个体 |

> **红焰当前鱼灵颜色：红**

### 能力边界

- **不能**直接看到数值、具体技能名、战斗数值
- **不能**穿透伪装（品种误判不影响颜色感知）
- 有代价：头痛、耳鸣、视野重影、精神疲惫
- **不能**无限开启，有冷却和精力消耗
- 陆沉当前处于能力不完整阶段，不能写成万能扫描仪

### 叙事价值

- 信息差的核心来源
- 红焰被严重低估的底层逻辑
- "低评分高潜力"在竞技层面的合法支撑

## 三、第二阶段：鱼技

### 一句话定义

> 陆沉能看出一条鱼"会什么"，而不只是"强不强"。

### 解锁条件

> 在与同一条红品以上鱼建立稳定羁绊后，亲历一次真正的正式竞技或生死级实战，并成功凭"理解鱼"而不是凭运气赢下关键节点，鱼技觉醒。

**三个好处：**
1. 不会太早开挂
2. 和红焰绑定，强化主角与鱼的羁绊
3. 觉醒理由是"第一次真正懂鱼"，而不是系统白送

### 能做什么

- 能识别一条鱼**已经掌握的技能类型**
- 能模糊判断技能属于：攻击 / 防御 / 控制 / 机动 / 辅助 / 被动
- 能看到某条鱼身上残留的**技能痕迹**
  - 比如它刚用过加速
  - 刚闭鳞隐匿
  - 曾经历过某类控制
- 能大致判断：这是"天生技能"还是"后天磨出来的技能"
- 对熟悉的鱼（尤其红焰），可以更精准地看出：技能名字倾向 / 技能强度 / 技能是否接近进阶

### 不能做什么

- 不能直接看到**完整技能说明书**
- 不能知道技能的全部数值和冷却时间
- 不能百分百分辨所有技能
- 面对陌生高品级鱼，只能看到"轮廓"和"类型"，看不全细节
- 不能复制技能，也不能赋予技能
- 对低品鱼/白板鱼，看到的信息非常少

### 过渡事件

> 陆沉在一场真正的斗鱼对抗里，第一次靠"红焰会飞跃/会隐匿/会背水"的理解，打出一波非运气型逆转。

也就是说，不是训练里觉醒，而是：
- 比赛中
- 被逼到墙角
- 他第一次不是凭"感觉红焰强"，而是明确意识到：**"红焰不是只有蛮力，它是有技能结构的。"**
- 就在这一刻，能力升级到「鱼技」

### 叙事价值

- 从"捡到怪鱼"升级成"开始像训练师"
- 比赛会明显变聪明
- 主角第一次拥有"战术理解力"而不是单纯奇遇

## 四、第三阶段：鱼态

### 一句话定义

> 陆沉能看出一条鱼"现在怎么样"，而不只是"它本来有多强"。

### 解锁条件

> 陆沉在连续高压比赛或训练中，必须面对"明明鱼很强，却在错误时机出手导致崩盘"的惨痛教训，随后才觉醒鱼态。

最适合放在：
- 蓝鲸杯后半
- 或龙门杯市级赛初段

因为这时候比赛开始真正需要"时机判断"。

### 能做什么

- 能读取鱼的实时状态：体力高低 / 伤势轻重 / 精神紧绷度 / 技能是否刚消耗过 / 是否进入疲劳期
- 能判断一条鱼是：虚张声势 / 强弩之末 / 正在蓄力 / 已经乱了
- 能感知某条鱼此刻更适合：追击 / 防守 / 潜伏 / 撤退
- 对己方鱼的帮助极大：什么时候该换鱼 / 什么时候该让红焰冲 / 什么时候必须拖时间

### 不能做什么

- 不能直接治伤
- 不能替鱼恢复体力
- 不能改写状态，只能读取状态
- 不能完全穿透伪装——有些善于藏状态的鱼仍会误导他
- 对从未接触过、等级太高的鱼，读取可能延迟或失真
- 一次关注太多鱼，会让陆沉自己极度疲惫

### 过渡事件

> 陆沉在一场关键战里，明明判断对了鱼技，却因为没看懂"红焰已经到极限"，差点把红焰打废。

这一下必须**疼**。因为鱼态这个能力，本质上不是奖励，是教训：
- 你光知道鱼会什么，不够
- 你还得知道它现在还能不能做

### 叙事价值

- 战斗不再只拼爆发，而开始拼节奏
- 主角开始像"临场指挥者"
- 比赛爽点会从"以弱胜强"升级到"抓窗口反杀"

## 五、第四阶段：鱼意

### 一句话定义

> 陆沉能感知鱼的情绪、欲望、战意和选择倾向，真正开始"懂鱼心"。

### 解锁条件

> 陆沉必须先与某条鱼建立绝对信任，并在一次"鱼自己做出选择"的重大事件中，真正承认鱼不是工具，而是有意志的生命，鱼意才会觉醒。

**一定不能太早**，因为它会把主角和鱼的关系从"主人+战宠"推进到"伙伴+战友"。

最适合的触发前提是：
- 红焰主动救他
- 或红焰违抗他的错误命令，却反而救了全局
- 陆沉第一次真正理解：**鱼不是听话的武器，而是有自己的战斗意志。**

### 能做什么

- 能读取鱼的情绪和战斗倾向：愤怒 / 恐惧 / 犹豫 / 饥饿 / 杀意 / 保护欲
- 能感知一条鱼到底想：战 / 退 / 藏 / 救队友 / 拼命
- 能更准确地判断：这条鱼会不会临阵崩 / 会不会背水一战 / 会不会护主
- 对高灵智鱼尤其有效
- 与红焰之间可以形成接近"无声默契"的配合

### 不能做什么

- 不能直接读完整思想
- 不能像读人类语言一样逐字逐句理解
- 不能强行操控意志
- 面对敌方高灵智鱼，只能感知意志方向，不能强制压制
- 情绪越复杂、灵智越高的鱼，读取越吃力
- 如果陆沉自身心态乱，读取结果也会失真

### 过渡事件

> 红焰在一场关键战里，违背陆沉表面指令，却做出了更正确的选择；陆沉先是愤怒，随后在生死一线中第一次真正明白"红焰有自己的意志"。

### 叙事价值

- 红焰的人格感会大幅增强
- 主角和鱼的关系真正升维
- 后续写"牺牲、保护、复仇、带队"都会更有重量

## 六、第五阶段：鱼视

### 一句话定义

> 陆沉能短暂借鱼为眼，从"人类视角"进入"鱼类战场视角"。

### 解锁条件

> 陆沉必须在极高压的正式比赛中，与红焰完成一次真正意义上的"心意同步"，并承担巨大精神代价，鱼视才会开启。

这个阶段必须是**高阶能力，不能在卷一早开**，因为它一开，战场理解会质变。

### 能做什么

- 短暂共享己方鱼的视角
- 在水下直接看到：地形 / 暗流 / 盲区 / 敌鱼位置 / 埋伏路线
- 能从鱼的运动逻辑理解战场
- 能在复杂地形战中获得巨大战术优势
- 对多鱼协同时，能进行极短时间的"视角切换"

### 不能做什么

- 不能长期开启
- 不能无限多线程看所有鱼
- 不能毫无代价

**开启过久会导致：**
- 眩晕
- 恶心
- 脑内信息过载
- 分不清自己和鱼的方向感

- 初期只能共享己方最有羁绊的鱼
- 无法稳定共享敌方鱼视角
- 在强干扰场地里（暗流、极端温差、感知混乱场），会非常危险

### 过渡事件

> 在一场必须靠地形和预判取胜的高阶战里，陆沉人类视角已经完全失效，红焰却在水下找到唯一生路；陆沉在极端压力中与红焰短暂同频，第一次"看见"红焰看见的世界。

这一下要写成**震撼场面**。因为鱼视不是普通升级，而是：
- 主角第一次真正跨进鱼的世界
- 也是整条能力线的质变点

### 叙事价值

- 战斗场景会立体化
- 地形战会非常好写
- 主角会真正从"训练师"变成"顶级指挥者"

## 七、阶段边界总表

| 阶段 | 看见什么 | 核心价值 | 最大限制 |
|------|----------|----------|---------|
| 鱼色 | 上限/品级 | 识强弱 | 只知潜力，不知细节 |
| 鱼技 | 技能/招式 | 懂手段 | 不知实时状态 |
| 鱼态 | 体力/伤势/窗口 | 抓时机 | 不知深层意志 |
| 鱼意 | 情绪/战意/选择倾向 | 懂鱼心 | 不能完全读思想 |
| 鱼视 | 水下视角/战场结构 | 控全局 | 代价极大，不能常开 |

---

## 八、能力边界总原则

1. **不能抄答案**：不能直接看到完整技能说明书/数值/冷却时间
2. **代价真实**：每一阶段都有头痛/眩晕/精神疲惫等代价
3. **信息差合理**：颜色感知是底层异能，竞技评分是官方体系，两者分离
4. **成长有代价**：越往后越强，但精神消耗也越大
5. **阶段性锁死**：必须解锁前置阶段，才能进入下一阶段
6. **卷一只开第一阶段**：后续四阶段是长线伏笔，不得在卷一提前解锁
