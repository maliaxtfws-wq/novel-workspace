## 三、赛事方

### 势力定义

赛事方指斗鱼竞技公开赛事体系的明面执行机构。它不是国家暴力机关，也不是秘密组织。它的功能是：

> 提供一套正式、公开、可量化、可晋级的竞技秩序。

### 已定硬锚点

赛事方负责：赛事审查、实名登记、战鱼校验、赛前战力检测、检录、抽签、场地执行、超级计算机保护机制、晋级与认证记录。

但它**不负责处理场外组织恩怨**，也不等于国家秘密项目本身。

### 赛事方内部层级

**A. 公开运营层**：公告页、报名点、检录员、赛事主持/解说/宣发、数据记录系统

**B. 规则执行层**：战鱼校验、评分系统、认证锁定、超级计算机保护机制、抽签与赛程系统

**C. 上层观察层**：只有极少数高层知道蓝鲸杯不只是比赛，也是在筛人。卷一后段再慢慢抬。

### 与其他势力关系

| 对象 | 关系 |
| ---- | ---- |
| 民间鱼圈 | 吸纳、筛选、规训 |
| 自然会 | 被渗透/被利用的对象，不是同阵营 |
| 国家斗王计划 | 明面窗口与人才入口 |
| 蓝星公司线 | 仅在参赛资格与公司内部名额层面产生交集 |
| 主角团 | 从低估到关注，再到重点观察 |

## 四、民间鱼圈

### 势力定义

民间鱼圈不是一个统一组织，而是一套松散但真实存在的社会网络，包含：钓鱼圈、水族街、鱼市、渔具店、浅圳本地民间选手、半职业俱乐部、私下交易/假证/替鱼/情报买卖等灰色边缘。

### 内部结构

**A. 野钓/河沟圈**：陆沉与陈海最初靠近的地方，不正规但最自由，是红焰登场世界，更接近"人和鱼"的原始关系

**B. 水族街/老店圈**：周老板的渔具店、老水族街这种熟人社会，老玩家老养鱼人懂行商户聚集，父亲旧线最容易从这里长出来

**C. 鱼市圈**：东港旧鱼市这种半正规半野路子的交易地，捡漏/试鱼/吹牛/直播/砍价真假混在一起

**D. 半职业俱乐部圈**：浪潮斗鱼俱乐部就是这一层代表，比民间散圈更专业，比真正顶级俱乐部更接地气

**E. 民间选手圈**：黄子豪这种本地小有名气选手，网红型/吹牛型/真正有两把刷子的草根型都混在一起

### 核心节点

1. 周老板的渔具店——主角团最初物理落脚点，旧江湖入口
2. 老水族街——老圈子、人情、旧时代味道
3. 东港旧鱼市——资源、捡漏、真假混杂
4. 浪潮斗鱼俱乐部——民间与正式体系之间的桥

## 五，自然会

### 势力定义

自然会的正式定位：末日教派组织/场外反派势力/贯穿全书的核心反派。它不是赛事官方，没有公开执法权，也不是单纯黑帮。它是有信仰、有目标、有长期计划的极端主义组织。

### 核心信念

> 渊眼洞窟中封印的巨兽是被封存的灾厄，释放它可以让世界重新洗牌。

所以它的根本立场不是"想赢比赛"，而是阻止国家斗王计划、破坏民间高手进入正式筛选体系、放任甚至推动洞窟危机恶化。

### 行动逻辑（卷一阶段）

**A. 监视**：先看谁是异常鱼，先看谁是异常能力者，不急着摊牌

**B. 试探**：匿名举报、伪装买家/记者/赞助人、提供假情报诱导主角犯错

**C. 渗透**：混进鱼市/水族圈/媒体圈/赛事外围，找机会接触陆沉红焰相关战鱼

**D. 筛选**：不是所有异常者都立刻下手，会判断这个人能拉拢/能利用/还是必须除掉

### 卷一当前状态

- 已埋：自然会监视已埋
- 未完全公开：组织结构、核心人物、真正执行层、与官方项目的具体博弈方式
- 卷一该做到的程度：先让读者知道有这么个东西在看陆沉，但不要太早写成正面打Boss

## 六、官方/国家项目

### 核心：国家「斗王」计划

成立于2020年。表面目的是研究洞窟生态，真实目的是：

> 选拔全国最优秀的养鱼人和鱼，从旁道进入洞窟，修复空气墙或处理千米级巨兽。

### 渊眼洞窟危机

- 南海大陆架边缘存在渊眼洞窟
- 水下200米处有"空气墙"
- 淡水鱼是唯一可以穿透空气墙的例外
- 洞窟内矿物可强化生物本能，也会导致鱼类狂暴、觉醒技能、部分人类基因变异
- 2018年事故后，国家封闭洞窟入口
- 洞窟深处存在千米级超级生物

### 为什么斗鱼大赛会公开化

> 国家需要用公开赛事从民间筛选真正有天赋的人和鱼。蓝鲸杯50万奖金本质上是国家扔出来吸高手和异常鱼的诱饵。

### 与赛事方关系

- 赛事方 = 明面执行系统
- 斗王计划 = 上层真实目的
- 不是所有赛事工作人员都知道斗王计划，只有少数更高层观察者知道

## 七、公司线人物关系

### 蓝星游戏（公司本体）

定位：浅圳一线大厂/大平台式公司，看起来体面，实际层层甩锅/压榨/冷处理/绩效逻辑极强。是"都市打工人困境"的制度代表。

### 关键人物

**张海明**
- 身份：蓝星游戏研发二组主管，陆沉直属领导，公司线当前最大现实反派
- 真正动机：贿赂上级100万拿项目，最后25万尾款被妻子赌博输光，急需50万堵窟窿，所以把希望压到蓝鲸杯奖金上

**赵洋**
- 身份：比赛线正式强敌，张海明表弟，蛊王体系持有者
- 与公司线的关系：他本人不在蓝星上班，但通过张海明这条线被公司利益和比赛利益硬绑在一起，参赛既是报恩也是替张海明赌命
- 关键意义：他是公司线和比赛线真正连起来的那颗钉子

**赵鹏**
- 身份：蓝星内部背刺者，看过陆沉电脑/本地分支/接过功劳
- 功能：证明"伤陆沉的不只是领导，还有体系里愿意踩着别人往上爬的人"

**林宁**
- 身份：HR/员工关系专员，不站陆沉不站张海明只站公司风险控制
- 功能：把口头辞职正式收束成"干净离职"，代表公司系统不是单一坏人，而是一套只关心风险和止损的机制

### 公司线关系总逻辑

```
高层领导（项目利益）
  ↓施压
张海明（主管/贿赂链断裂/需50万）
  ├─压榨陆沉
  ├─利用赵鹏
  ├─借HR系统止损（林宁）
  └─资源向赵洋倾斜
赵洋（比赛线强敌/表弟/报恩参赛）
```

## 八、五大势力关系

| 关系 | 说明 |
| ---- | ---- |
| 赛事方 ←→ 民间鱼圈 | 吸纳、审查、筛选、规训——把散兵游勇变成有正式晋级可能的秩序竞技群体 |
| 赛事方 ←→ 国家斗王计划 | 表层窗口 vs 上层真实目的——赛事方是舞台，斗王计划是筛人机器 |
| 国家斗王计划 ←→ 自然会 | 根本对立——一个要修补危机筛选战力，一个要放大危机破坏筛选 |
| 民间鱼圈 ←→ 自然会 | 可渗透、可伪装、可利用——民间圈越复杂越适合自然会藏身 |
| 公司线 ←→ 比赛线 | 现实压迫线与竞技对抗线的接口——没有公司线陆沉不会这么快被逼上赛道 |

## 十、写作使用原则

1. **赛事方 ≠ 国家斗王计划**：赛事方是表层秩序，斗王计划是上层真实目的，必须分开
2. **自然会 ≠ 官方反派**：自然会不是官方审查员/裁判/认证官，它是场外反派组织
3. **民间鱼圈不是纯好人联盟**：这里有人情也有灰色地带，也有黑市/假证/替鱼/情报买卖，这样才真实
4. **公司线要尽快从"纯职场斗争"升级成"旧人生的债"**：赵洋一定重要，因为他是让张海明从"可恨领导"升级成"把现实压迫狠狠干进赛场的人"
